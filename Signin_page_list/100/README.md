# 시안 100 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 다크 아카이브  
- **UI 구조:** 단일 포커스 입력 흐름  
- **인터랙션:** GSAP 기반 부드러운 전환  

---

## 콘셉트

어두운 아카이브 톤의 **다크 아카이브** 무드에, **단일 포커스 입력 흐름**(한 번에 하나의 입력만 노출·다음/이전으로 단계 이동)과 **GSAP 기반 부드러운 전환**(스텝 전환 시 페이드·포커스 이동)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **다크 아카이브:** 배경 #141312, 서피스 #1c1a18, Crimson Pro 이탤릭 타이틀·Source Sans 3 본문, 밝은 그레이 액센트 #e7e5e4, 풀스크린 고정(overflow hidden).
- **단일 포커스 입력 흐름:** 4단계(이메일 → 이름 → 비밀번호 → 비밀번호 확인). 현재 단계만 표시, 다음/이전 시 GSAP 페이드 후 다음 입력으로 포커스 이동.
- **GSAP 기반 부드러운 전환:** 스텝 전환 시 `.s100-step--active` opacity 0 → step 변경 → opacity 1, 레이아웃 초기 등장 from opacity 0 / y 10.
- **중앙 정렬:** `.signup-fixed` + `.s100-layout`으로 뷰포트 가로·세로 중앙, max-width 360px, 스크롤 없는 고정 플로우.
- **반응형:** clamp·100dvh·단일 컬럼으로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 단계별 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·단일 컬럼으로 PC·모바일 전 범위 대응.
