## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 현대 판타지  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** 입력 반응형 애니메이션  

---

## 콘셉트

딥 퍼플·앰버 골드 톤의 **현대 판타지** 무드에, **카드 분리형 레이아웃**(기본 정보 카드 / 계정 보안 카드)과 **입력 반응형 애니메이션**(포커스 시 테두리·그림자·라벨 색 전환)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **현대 판타지:** 배경 그라데이션(#1a1625→#2d2438→#1f1b2e), 앰버·퍼플 글로우, Cormorant Garamond·Outfit, 글래스 카드(backdrop-filter), 액센트 #d4af37.
- **카드 분리형 레이아웃:** PC에서 2열 그리드 — 좌측 카드「기본 정보」(이메일·이름), 우측 카드「계정 보안」(비밀번호·비밀번호 확인·가입하기). 모바일에서는 1열로 세로 배치.
- **입력 반응형 애니메이션:** 포커스 시 `.s90-field--focus`로 테두리·박스섀도우·라벨 색이 앰버로 전환, blur 시 원복. 에러 시 라벨·테두리 빨간색 강조.
- **중앙 정렬:** `.signup-fixed` + `.s90-layout`으로 뷰포트 가로·세로 중앙, max-width 720px(레이아웃)·680px(카드 그리드) 반응형.
- **반응형:** 640px 이하에서 카드 1열, 헤더 상단·카드 하단.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.
