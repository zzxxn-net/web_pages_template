# 회원가입 페이지 템플릿 시안 60

미래적 클린 톤의 카드 분리형 회원가입 페이지 MVP 시안입니다.  
각 입력 영역을 카드로 분리하고, 포커스·유효·에러 상태를 시각적으로 강조합니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 미래적 클린  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** 상태 변화 강조  

## 콘셉트 및 UI/UX 특징

- **미래적 클린:** 밝은 배경(#f0f4f8)·DM Sans·Outfit, 시안 액센트(#0ea5e9)로 깔끔하고 미니멀한 톤을 줍니다.
- **카드 분리형 레이아웃:** 이메일·이름·비밀번호·비밀번호 확인을 각각 하나의 카드로 분리해 정보 단위를 명확히 합니다.
- **상태 변화 강조:** 카드 단위로 idle / focus / valid / invalid 상태를 테두리·그림자·라벨 색상으로 구분하고, 유효 시 "확인됨·일치" 문구를 노출하며, GSAP으로 포커스 시 카드 scale 1.01 전환을 적용해 상태 변화를 강조합니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(검증·제출·cardState·getFieldValid·GSAP 등장·카드 포커스 scale)  
- **동작:** Alpine.js로 focusedField·errors·cardState·getFieldValid·검증·제출, GSAP으로 `.s60-wrap` 등장·카드 포커스 시 scale 애니메이션.

## 중앙 정렬 및 반응형

- 모든 해상도에서 `.s60-wrap`을 가로·세로 중앙에 배치하고, 폼과 카드는 중앙 정렬을 유지합니다.  
- `max-width: 420px`·패딩으로 PC~모바일 전 범위 반응형을 적용했습니다.
