# 시안 75 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 사이버 미니멀  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** GSAP 기반 부드러운 전환  

---

## 콘셉트

다크 톤의 사이버 미니멀 무드에, 입력 필드별로 **카드를 분리**해 한 블록씩 명확히 구분하는 회원가입 폼이다.  
GSAP으로 래퍼와 카드가 순차 등장하며, 포커스 시 카드 테두리·라벨 색상이 부드럽게 전환되도록 했다.

## UI/UX 특징

- **카드 분리형:** 이메일 / 이름 / 비밀번호 / 비밀번호 확인을 각각 `.s75-card`로 감싸 카드 단위로 시각적 분리.
- **중앙 정렬:** `.signup-fixed` + `.s75`로 뷰포트 기준 가로·세로 중앙 정렬, PC~모바일 반응형 유지.
- **포커스 피드백:** 포커스 시 `s75-card--focused`로 테두리·글로우·라벨 색상(블루 액센트) 전환.
- **에러 상태:** 유효성 검사 실패 시 `s75-card--invalid`로 테두리 색상 강조.
- **GSAP:**  
  - 페이지 로드 시 `.s75-wrap` 페이드인 + 약간의 위로 이동.  
  - `.s75-card`들은 stagger로 짧은 딜레이를 두고 순차 등장.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 입력 + 클라이언트 유효성 검사 + 제출 버튼.
- **기술 스택:** HTMX + Alpine.js + TypeScript(js/main.ts → main.js) + SASS(style.scss → style.css) + GSAP.
- **파일 구조:**
  - `index.html`: 메인 마크업, Alpine `x-data="signupForm75()"`, HTMX 폼, GSAP CDN.
  - `style.scss` / `style.css`: 시안 75 전용 스타일(다크 배경, IBM Plex Mono / Space Grotesk, 블루 액센트).
  - `js/main.ts`: Alpine 컴포넌트 `signupForm75`, 유효성 검사, 제출 처리.
  - `js/main.js`: main.ts의 브라우저 실행용 번들(타입 제거).

## 반응형

- `clamp(1rem, 3vw, 2rem)` 패딩, `max-width: 400px` 래퍼로 PC·태블릿·모바일에서 동일한 중앙 정렬 플로우 유지.
- `100dvh` 사용으로 모바일 주소창 높이 변화에 대응.
