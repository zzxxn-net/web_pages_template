# 시안 117 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 다크 아카이브  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** 상태 변화 강조  

---

## 콘셉트

어두운 배경(#14171c)·골드 액센트(#c9a227)·IBM Plex Serif/Sans 기반의 **다크 아카이브** 무드에, **카드 분리형 레이아웃**(각 입력 필드를 개별 카드로 분리)과 **상태 변화 강조**(포커스·유효·오류 시 카드 테두리·배경·글로우로 뚜렷한 시각 전환)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **다크 아카이브:** 배경 #14171c, 카드 #1f242d, 텍스트 #e2e6ed, 골드 #c9a227, IBM Plex Serif(제목)·Sans(본문).
- **카드 분리형:** 이메일·이름·비밀번호·비밀번호 확인을 각각 `.s117-card`로 감싸 시각적으로 분리.
- **상태 변화 강조:** 포커스 시 카드 테두리·box-shadow 골드; 유효 시 녹색 테두리·배경·라벨 색; 오류 시 빨간 테두리·배경·에러 메시지. transition으로 부드럽게 전환.
- **중앙 정렬:** 뷰포트 가로·세로 중앙, max-width 420px.
- **반응형:** clamp·100dvh로 PC~모바일 전 범위 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·max-width 420px로 PC·모바일 전 범위 대응.
