# 회원가입 페이지 템플릿 시안 53

다크 아카이브 톤의 단일 포커스 입력 흐름 회원가입 페이지 MVP 시안입니다.  
한 필드씩 포커스하여 입력하고, 유효/무효·포커스 상태를 시각적으로 강조했습니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 다크 아카이브  
- **UI 구조:** 단일 포커스 입력 흐름  
- **인터랙션:** 상태 변화 강조  

## 콘셉트 및 UI/UX 특징

- **다크 아카이브:** 다크 배경(#0f0f0f)·노이즈 텍스처, IBM Plex Serif·Mono 폰트, 골드 포인트(#c9a227)로 아카이브/문서 감성을 줍니다.
- **단일 포커스 입력 흐름:** 이메일·이름·비밀번호·비밀번호 확인을 한 화면에 두되, 포커스된 필드만 테두리·그림자로 강조해 한 번에 하나의 입력에 집중하도록 했습니다.
- **상태 변화 강조:** 유효 시 ✓(녹색)·테두리, 무효 시 !(빨강)·테두리·에러 문구, 포커스 시 골드 링·배경 미세 강조로 상태 변화를 명확히 보여줍니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(가입 폼 검증·제출·포커스 상태)  
- **동작:** Alpine.js로 이메일·이름·비밀번호·비밀번호 확인 검증 및 제출, 포커스 시 필드 강조(CSS), GSAP으로 카드 등장 애니메이션.

## 중앙 정렬 및 반응형

- 모든 해상도에서 `.s53-wrap`을 가로·세로 중앙에 배치하고, 카드와 폼은 중앙 정렬을 유지합니다.  
- `max-width: 400px`·패딩으로 PC~모바일 전 범위 반응형을 적용했습니다.
