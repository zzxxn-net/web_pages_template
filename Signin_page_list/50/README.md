# 회원가입 페이지 템플릿 시안 50

감성 테크 톤의 비대칭 레이아웃 회원가입 페이지 MVP 시안입니다.  
PC·모바일 전 범위 반응형, 스크롤 없는 고정 플로우로 한 화면에서 가입을 완료할 수 있도록 구성했습니다.

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** 스크롤 없는 고정 플로우  

## 콘셉트 및 UI/UX 특징

- **감성 테크:** 따뜻한 베이지·코랄 계열 그라데이션, 블러 블롭 배경, Noto Sans KR·Outfit 폰트로 감성적이면서도 읽기 쉬운 톤을 유지합니다.
- **비대칭 레이아웃:** 왼쪽에 웰컴 문구·태그라인·데코레이션을 두고, 오른쪽에 폼 카드를 배치해 시선 흐름을 분리했습니다. 모바일에서는 상단 문구·하단 폼 순으로 세로 배치됩니다.
- **스크롤 없는 고정 플로우:** 뷰포트 전체를 사용하되 `overflow: hidden`으로 스크롤을 막고, 필수 입력 필드만 포함해 한 화면에서 가입이 완료되도록 했습니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(가입 폼 검증·제출·포커스 상태)  
- **동작:** Alpine.js로 이메일·이름·비밀번호·비밀번호 확인 검증, 포커스 시 필드 강조, GSAP으로 좌측 문구·우측 폼 섹션 등장 애니메이션 적용.

## 중앙 정렬 및 반응형

- 데스크톱: 그리드 2열(좌 문구 / 우 폼), 각 영역 가로·세로 중앙 정렬.  
- 900px 이하: 1열로 전환, 상단 문구·하단 폼 순, 동일하게 중앙 정렬 유지.
