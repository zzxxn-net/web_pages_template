# 회원가입 페이지 템플릿 시안 52

사이버 미니멀 톤의 단계 분할형(Progressive Form) 회원가입 페이지 MVP 시안입니다.  
단계별로 한 필드씩 진행하며, GSAP으로 패널 전환을 부드럽게 연출했습니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 사이버 미니멀  
- **UI 구조:** 단계 분할형(Progressive Form)  
- **인터랙션:** GSAP 기반 부드러운 전환  

## 콘셉트 및 UI/UX 특징

- **사이버 미니멀:** 다크 배경·그리드 패턴, Inter·JetBrains Mono 폰트, 그린 포인트 컬러(#34d399)로 미니멀하고 테크한 분위기를 줍니다.
- **단계 분할형(Progressive Form):** 이메일(1) → 이름(2) → 비밀번호(3) → 비밀번호 확인(4) 순으로 한 단계씩 노출하며, 상단 스텝 버튼으로 단계 이동·현재 단계 표시를 합니다.
- **GSAP 기반 부드러운 전환:** 다음/이전/스텝 클릭 시 현재 패널은 opacity·y로 퇴장하고, 다음 패널은 fade·slide로 등장하도록 GSAP으로 처리했습니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(단계·검증·goNext/goPrev/goToStep·GSAP 패널 전환)  
- **동작:** Alpine.js로 step 상태·필드 검증·제출, GSAP으로 wrap 초기 등장·단계 전환 시 패널 아웃/인 애니메이션.

## 중앙 정렬 및 반응형

- 모든 해상도에서 `.s52-wrap`을 가로·세로 중앙에 배치하고, 카드와 폼은 중앙 정렬을 유지합니다.  
- `max-width: 420px`·패딩으로 PC~모바일 전 범위 반응형을 적용했습니다.
