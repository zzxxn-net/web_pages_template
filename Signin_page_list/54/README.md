# 회원가입 페이지 템플릿 시안 54

미래적 클린 톤의 풀스크린 몰입형 회원가입 페이지 MVP 시안입니다.  
Enter로 다음 필드로 포커스를 이동하는 포커스 이동 중심 UX를 적용했습니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 미래적 클린  
- **UI 구조:** 풀스크린 몰입형  
- **인터랙션:** 포커스 이동 중심 UX  

## 콘셉트 및 UI/UX 특징

- **미래적 클린:** 밝은 배경(#f8fafc)·Inter 폰트·스카이 블루 포인트(#0ea5e9)로 미니멀하고 미래적인 톤을 줍니다.
- **풀스크린 몰입형:** 뷰포트 전체를 사용하고, 사이드바·카드 틀 없이 헤더와 폼만 중앙에 배치해 몰입형 레이아웃으로 구성했습니다.
- **포커스 이동 중심 UX:** 이메일 → 이름 → 비밀번호 → 비밀번호 확인 순으로 Enter 시 다음 필드로 포커스가 이동하고, 마지막 필드에서 Enter 시 가입 제출이 되도록 했습니다. 포커스된 필드는 테두리·그림자로 강조됩니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(검증·focusNext·폼 제출·GSAP 등장)  
- **동작:** Alpine.js로 검증·focusedField·focusNext(Enter 시 다음 필드 포커스)·제출, GSAP으로 `.s54-full` 등장 애니메이션.

## 중앙 정렬 및 반응형

- 모든 해상도에서 `.s54-full`을 가로·세로 중앙에 배치하고, 폼은 중앙 정렬을 유지합니다.  
- `max-width: 440px`·패딩으로 PC~모바일 전 범위 반응형을 적용했습니다.
