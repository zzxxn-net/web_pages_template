# 회원가입 페이지 템플릿 시안 67

현대 판타지 톤의 풀스크린 몰입형 회원가입 페이지 MVP 시안입니다.  
스크롤 없이 한 화면에 고정하여 입력·제출이 모두 보이도록 했습니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 현대 판타지  
- **UI 구조:** 풀스크린 몰입형  
- **인터랙션:** 스크롤 없는 고정 플로우  

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 따뜻한 다크 배경(#1a1612)·Crimson Pro·Literata, 골드/앰버 액센트(#c9a227)로 판타지 감성을 줍니다.
- **풀스크린 몰입형:** 뷰포트 전체(min-height 100dvh)를 배경으로 채우고, 헤더·폼만 중앙(.s67-full)에 배치해 몰입형 레이아웃으로 구성했습니다.
- **스크롤 없는 고정 플로우:** overflow: hidden·max-width 360px·간격 축소(0.7rem·0.55rem 패딩)로 스크롤 없이 한 화면에 모든 입력·제출 버튼이 들어가도록 했습니다. GSAP으로 .s67-full 등장(opacity·y) 애니메이션을 적용했습니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(검증·제출·GSAP 등장)  
- **동작:** Alpine.js로 focusedField·errors·검증·제출, GSAP으로 .s67-full 등장.

## 중앙 정렬 및 반응형

- .s67-full은 뷰포트 가로·세로 중앙에 배치하고, 폼은 중앙 정렬을 유지합니다.  
- max-width 360px·패딩으로 PC~모바일 전 범위 반응형을 적용했으며, 스크롤 없이 고정 플로우가 유지됩니다.
