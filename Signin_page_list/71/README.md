# 시안 71 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 네오 브루탈리즘
- **UI 구조:** 마이크로 인터랙션 중심
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트

네오 브루탈리즘 톤과 마이크로 인터랙션, 입력 반응형 애니메이션을 결합한 회원가입 MVP 시안입니다. 두꺼운 테두리·노출된 그리드·강한 타이포로 브루탈한 인상을 주고, 포커스·호버·에러 시 작은 피드백과 GSAP 입력 애니메이션으로 반응을 강조합니다.

## UI/UX 특징

- **네오 브루탈리즘:** 4px 검정 테두리, 8px 오프셋 박스 섀도우, Archivo Black·Space Mono, 노란 배경 버튼(#ff0), 밝은 배경(#f5f5f0)
- **마이크로 인터랙션 중심:** 포커스 시 필드 왼쪽 보더·라벨 색 변화, 호버 시 SHOW 버튼 역전, SUBMIT 호버 시 translate·shadow 감소, 에러 시 입력창 배경·테두리 강조
- **입력 반응형 애니메이션:** GSAP으로 .s71-box 등장, 입력 포커스 시 scale 1.02·블러 시 1로 복귀

## 사용 의도 및 구조

- **index.html:** 메인 마크업, Alpine.js `signupForm()` 바인딩, HTMX 속성, s71-box·s71-field 구조
- **style.scss / style.css:** 시안 전용 스타일, 반응형 clamp
- **js/main.ts, js/main.js:** 검증·제출·GSAP 등장·포커스 scale
- PC~모바일 전 범위 중앙 정렬·반응형 대응
