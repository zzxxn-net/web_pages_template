# 시안 91 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크  
- **UI 구조:** 단계 분할형(Progressive Form)  
- **인터랙션:** 포커스 이동 중심 UX  

---

## 콘셉트

따뜻한 크림·블러시 톤의 **감성 테크** 무드에, **단계 분할형(Progressive Form)**(1단계 기본 정보 → 2단계 계정 보안)과 **포커스 이동 중심 UX**(다음/이전 클릭 시 해당 단계 첫 입력 필드로 포커스 이동)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **감성 테크:** 배경 그라데이션(#fef7f4→#f5eae6), Noto Sans KR, 부드러운 카드 그림자, 액센트 #c97b63(테라코타).
- **단계 분할형(Progressive Form):** 상단 단계 인디케이터(1·2), 1단계 패널(이메일·이름·다음), 2단계 패널(비밀번호·비밀번호 확인·이전·가입하기). 현재 단계만 표시, 전환 시 포커스 이동.
- **포커스 이동 중심 UX:** 「다음」 클릭 시 2단계로 전환 후 비밀번호 필드에 포커스, 「이전」 클릭 시 1단계로 전환 후 이메일 필드에 포커스.
- **중앙 정렬:** `.signup-fixed` + `.s91-layout`으로 뷰포트 가로·세로 중앙, max-width 420px 반응형.
- **반응형:** clamp·100dvh·단일 컬럼으로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 단계별 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh로 PC·모바일 전 범위 대응.
