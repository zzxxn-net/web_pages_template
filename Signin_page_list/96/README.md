# 시안 96 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 실험적 인터페이스  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** GSAP 기반 부드러운 전환  

---

## 콘셉트

다크 톤·퍼플 액센트의 **실험적 인터페이스** 무드에, **비대칭 레이아웃**(좌측 38% 브랜드·우측 62% 폼)과 **GSAP 기반 부드러운 전환**(로드 시 좌·우에서 각각 슬라이드 인)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **실험적 인터페이스:** 배경 #0c0c0c, 좌측 #171717·우측 #1a1a1a 분리, Syne 폰트, 퍼플 액센트 #a78bfa, 테두리 없는 그리드 블록.
- **비대칭 레이아웃:** PC에서 grid 0.38fr(좌)·0.62fr(우), 좌측 브랜드(Join·실험적인 한 걸음), 우측 폼(이메일·이름·비밀번호·비밀번호 확인·가입하기). 모바일에서는 1열 상단 브랜드·하단 폼.
- **GSAP 기반 부드러운 전환:** 로드 시 `.s96-layout` 페이드인 후, `.s96-aside` from(x: -20), `.s96-form-wrap` from(x: 20)으로 좌·우에서 부드럽게 등장.
- **중앙 정렬:** `.signup-fixed` + `.s96-layout`으로 뷰포트 가로·세로 중앙, max-width 880px(레이아웃)·320px(폼) 반응형.
- **반응형:** 640px 이하에서 1열, 브랜드 상단·폼 하단.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.
