# 시안 89 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 현대 판타지  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** GSAP 기반 부드러운 전환  

---

## 콘셉트

딥 퍼플·앰버 골드 톤의 **현대 판타지** 무드에, **비대칭 레이아웃**(좌측 브랜드·우측 폼 카드)과 **GSAP 기반 부드러운 전환**(로드 시 브랜드 좌측·폼 우측에서 각각 페이드인)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **현대 판타지:** 배경 그라데이션(#1a1625→#2d2438→#1f1b2e), 앰버·퍼플 글로우, Cormorant Garamond·Outfit, 글래스 카드(backdrop-filter), 액센트 #d4af37.
- **비대칭 레이아웃:** PC에서 좌측 브랜드(타이틀·태그라인), 우측 폼 카드(이메일·이름·비밀번호·비밀번호 확인·가입하기). 모바일에서는 상단 브랜드·하단 폼으로 단일 컬럼.
- **GSAP 기반 부드러운 전환:** 로드 시 `.s89-layout` 페이드인 후, `.s89-brand` from(x: -24), `.s89-form-section` from(x: 24)로 좌·우에서 부드럽게 등장.
- **중앙 정렬:** `.signup-fixed` + `.s89-layout`으로 뷰포트 가로·세로 중앙, max-width 900px(레이아웃)·380px(폼 카드) 반응형.
- **반응형:** 700px 이하에서 그리드 1열, 브랜드 상단·폼 하단.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.
