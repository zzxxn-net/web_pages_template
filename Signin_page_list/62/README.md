# 회원가입 페이지 템플릿 시안 62

네오 브루탈리즘 톤의 단일 포커스 입력 흐름 회원가입 페이지 MVP 시안입니다.  
스크롤 없이 한 화면에 고정하고, 한 번에 하나의 필드만 시각적으로 강조합니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 네오 브루탈리즘  
- **UI 구조:** 단일 포커스 입력 흐름  
- **인터랙션:** 스크롤 없는 고정 플로우  

## 콘셉트 및 UI/UX 특징

- **네오 브루탈리즘:** 검정 배경(#0a0a0a)·Archivo Black·IBM Plex Sans, 굵은 테두리(3px)·노란 액센트(#facc15)로 노출된 구조감과 강한 대비를 줍니다.
- **단일 포커스 입력 흐름:** 포커스된 필드만 활성(노란 테두리·라벨 색상)으로 강조하고, 나머지 필드는 `s62-field--dim`으로 opacity 0.45를 적용해 한 번에 하나의 항목에만 시선이 가도록 합니다.
- **스크롤 없는 고정 플로우:** 뷰포트를 `overflow: hidden`·`min-height: 100dvh`로 고정하고, 폼을 컴팩트하게(max-width 380px·간격 축소) 구성해 스크롤 없이 한 화면에 모든 입력·제출 버튼이 들어가도록 했습니다. GSAP으로 `.s62-frame` 등장 시 scale·opacity 전환을 적용합니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(검증·제출·GSAP 등장)  
- **동작:** Alpine.js로 focusedField·errors·검증·제출, GSAP으로 `.s62-frame` 등장 애니메이션.

## 중앙 정렬 및 반응형

- `.s62-frame`은 뷰포트 가로·세로 중앙에 배치하고, 내부 헤더·폼은 중앙 정렬을 유지합니다.  
- `max-width: 380px`·패딩으로 PC~모바일 전 범위 반응형을 적용했으며, 스크롤 없이 고정 플로우가 유지됩니다.
