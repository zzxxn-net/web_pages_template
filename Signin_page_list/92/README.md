# 시안 92 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 다크 아카이브  
- **UI 구조:** 풀스크린 몰입형  
- **인터랙션:** 스크롤 없는 고정 플로우  

---

## 콘셉트

스톤·차콜 톤의 **다크 아카이브** 무드에, **풀스크린 몰입형**(100dvh 뷰포트 전체 사용·중앙 정렬)과 **스크롤 없는 고정 플로우**(overflow hidden·컴팩트 간격으로 한 화면에 모든 입력·제출 노출)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **다크 아카이브:** 배경 #1c1917, 서피스 #292524, Lora·Source Sans 3, 스톤 액센트 #d6d3d1, 아카이브 감성 타이틀 "Archive".
- **풀스크린 몰입형:** main 높이 100dvh, flex 중앙 정렬로 폼이 뷰포트 전체에 몰입 배치.
- **스크롤 없는 고정 플로우:** `.s92--full`로 overflow: hidden, 컴팩트한 필드·간격으로 이메일·이름·비밀번호·비밀번호 확인·가입하기가 한 화면에 고정 노출.
- **중앙 정렬:** `.signup-fixed` + `.s92-layout`으로 가로·세로 중앙, max-width 380px 반응형.
- **반응형:** clamp·100dvh·컴팩트 패딩으로 PC·모바일 전 범위 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·overflow hidden으로 스크롤 없는 고정 플로우 유지.
