# 시안 83 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** 포커스 이동 중심 UX  

---

## 콘셉트

딥 블루·코랄 액센트·부드러운 글로우의 **감성 테크** 무드에, **비대칭 레이아웃**(좌측 브랜드·우측 폼 카드)과 **포커스 이동 중심 UX**(Enter 시 다음 필드 자동 포커스, 좌측에 현재 필드 힌트 표시)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **감성 테크:** 배경 그라데이션(#1a1a2e→#16213e→#0f3460), 코랄·티얼 글로우, Outfit·DM Sans, 반투명 카드·backdrop-filter, 액센트 #e94560·#4ecdc4.
- **비대칭 레이아웃:** PC에서 좌측 브랜드(타이틀·태그라인·포커스 힌트), 우측 폼 카드(이메일·이름·비밀번호·비밀번호 확인·가입하기). 모바일에서는 상단 브랜드·하단 폼으로 단일 컬럼.
- **포커스 이동 중심 UX:** 필드 포커스 시 해당 필드만 강조(라벨·테두리 색), 좌측에 "이메일을 입력해 주세요." 등 힌트 표시. Enter 시 유효성 통과하면 다음 필드로 자동 포커스. 마지막 필드 Enter 시 제출.
- **중앙 정렬:** `.signup-fixed` + `.s83-layout`로 뷰포트 기준 가로·세로 중앙, PC·모바일 반응형.
- **GSAP:** 로드 시 `.s83-layout` 페이드인 + 위로 이동.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- 720px 이하에서 그리드 1열, 브랜드 상단·폼 하단. clamp·100dvh·max-width 920px(레이아웃)·400px(폼 카드)로 PC·모바일 대응.
