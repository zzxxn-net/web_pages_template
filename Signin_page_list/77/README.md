# 시안 77 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 다크 아카이브  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** 포커스 이동 중심 UX  

---

## 콘셉트

어두운 아카이브·문서 톤의 **다크 아카이브** 무드에, **비대칭 그리드**(폼 열 + 비주얼 열)와 **Enter로 다음 필드로 이동**하는 **포커스 이동 중심 UX**를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **다크 아카이브:** 배경 #0c0a09, 그리드 텍스처, IBM Plex Mono·IBM Plex Sans KR, 악센트 #a78b71, 서피스 #161412.
- **비대칭 레이아웃:** `.s77-inner` 2열 그리드(1fr 1.15fr). 왼쪽 폼 영역, 오른쪽 비주얼(ARCHIVE 문구). 640px 이하에서 1열·비주얼 하단.
- **포커스 이동 중심 UX:** 각 입력에서 Enter 시 `focusNext()`로 다음 필드로 포커스 이동. 포커스 시 라벨·테두리 액센트 색·글로우로 시각적 강조.
- **중앙 정렬:** `.signup-fixed` + `.s77`로 뷰포트 기준 가로·세로 중앙, PC·모바일 반응형.
- **GSAP:** 로드 시 `.s77-inner` 페이드인 + 위로 이동.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html(비대칭 마크업, x-ref 입력 포커스 이동), style.scss → style.css, js/main.ts → main.js(focusNext·검증·GSAP), README.md.

## 반응형

- `max-width: 900px` 그리드, 640px 이하 1열·비주얼 상단 border-top. `clamp`·`100dvh`로 디바이스 대응.
