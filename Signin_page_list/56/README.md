# 회원가입 페이지 템플릿 시안 56

네오 브루탈리즘 톤의 풀스크린 몰입형 회원가입 페이지 MVP 시안입니다.  
GSAP으로 헤더·필드·버튼이 부드럽게 순차 등장하도록 연출했습니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 네오 브루탈리즘  
- **UI 구조:** 풀스크린 몰입형  
- **인터랙션:** GSAP 기반 부드러운 전환  

## 콘셉트 및 UI/UX 특징

- **네오 브루탈리즘:** 베이지 배경(#f5f5f0)·흑백 고대비, Archivo Black·Archivo 폰트, 굵은 테두리(3px)·각진 형태(border-radius 0), 버튼·입력 포커스 시 오프셋 그림자(4px 4px 0)로 브루탈한 톤을 줍니다.
- **풀스크린 몰입형:** 뷰포트 전체를 사용하고, 헤더와 폼만 중앙에 배치해 몰입형 레이아웃으로 구성했습니다.
- **GSAP 기반 부드러운 전환:** 초기 로드 시 헤더(위에서 fade·slide) → 필드(순차 stagger) → 제출 버튼(딜레이 후 등장) 순으로 부드럽게 등장하도록 GSAP으로 처리했습니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(검증·제출·GSAP 등장)  
- **동작:** Alpine.js로 검증·focusedField·제출, GSAP으로 헤더·필드·액션 영역 순차 등장 애니메이션.

## 중앙 정렬 및 반응형

- 모든 해상도에서 `.s56-full`을 가로·세로 중앙에 배치하고, 폼은 중앙 정렬을 유지합니다.  
- `max-width: 420px`·패딩으로 PC~모바일 전 범위 반응형을 적용했습니다.
