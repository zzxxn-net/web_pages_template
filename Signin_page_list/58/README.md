# 회원가입 페이지 템플릿 시안 58

현대 판타지 톤의 단일 포커스 입력 흐름 회원가입 페이지 MVP 시안입니다.  
GSAP으로 wrap·필드·버튼이 부드럽게 순차 등장하고, 포커스된 필드만 시각적으로 강조됩니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 현대 판타지  
- **UI 구조:** 단일 포커스 입력 흐름  
- **인터랙션:** GSAP 기반 부드러운 전환  

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 베이지 배경(#f6f2eb)·Cormorant Garamond·Noto Sans KR, 골드/앰버 포인트(#9a7b4f)로 판타지 감성과 가독성을 함께 줍니다.
- **단일 포커스 입력 흐름:** 이메일·이름·비밀번호·비밀번호 확인을 한 화면에 두되, 포커스된 필드만 scale(1.02)·라벨 색상·테두리·그림자로 강조해 한 번에 하나의 입력에 집중하도록 했습니다.
- **GSAP 기반 부드러운 전환:** 초기 로드 시 wrap(fade·slide) → 필드(stagger) → 제출 버튼(딜레이 후) 순으로 부드럽게 등장하도록 GSAP으로 처리했습니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(검증·제출·GSAP 등장)  
- **동작:** Alpine.js로 focusedField·검증·제출, GSAP으로 wrap·필드·액션 영역 순차 등장 애니메이션.

## 중앙 정렬 및 반응형

- 모든 해상도에서 `.s58-wrap`을 가로·세로 중앙에 배치하고, 폼은 중앙 정렬을 유지합니다.  
- `max-width: 400px`·패딩으로 PC~모바일 전 범위 반응형을 적용했습니다.
