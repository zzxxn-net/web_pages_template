# 시안 101 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 미래적 클린  
- **UI 구조:** 풀스크린 몰입형  
- **인터랙션:** 스크롤 없는 고정 플로우  

---

## 콘셉트

밝은 그레이·화이트·블루 액센트의 **미래적 클린** 무드에, **풀스크린 몰입형**(뷰포트 전체·overflow hidden)과 **스크롤 없는 고정 플로우**(한 화면에 모든 입력·가입하기 배치, 컴팩트 간격으로 스크롤 없이 완결)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **미래적 클린:** 배경 #f8fafc, DM Sans, 블루 액센트 #3b82f6, 포커스 시 2px 링·라벨 색 전환, 에러 시 빨간 테두리·라벨.
- **풀스크린 몰입형:** `.s101--full`로 height 100vh/100dvh, overflow hidden, 패딩 clamp로 여백 최소화.
- **스크롤 없는 고정 플로우:** max-width 340px, 필드·액션 간격 clamp(0.4rem~0.6rem)로 컴팩트 배치, 한 화면에 이메일·이름·비밀번호·비밀번호 확인·가입하기 모두 표시.
- **중앙 정렬:** `.signup-fixed` + `.s101-layout`으로 뷰포트 가로·세로 중앙, flex-shrink 0으로 레이아웃 고정.
- **반응형:** clamp·100dvh·단일 컬럼으로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·단일 컬럼으로 PC·모바일 전 범위 대응.
