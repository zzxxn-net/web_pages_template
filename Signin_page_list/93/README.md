# 시안 93 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 사이버 미니멀  
- **UI 구조:** 단일 포커스 입력 흐름  
- **인터랙션:** 상태 변화 강조  

---

## 콘셉트

쿨 그레이·시안 액센트의 **사이버 미니멀** 무드에, **단일 포커스 입력 흐름**(한 번에 한 필드만 노출·다음/이전으로 전환)과 **상태 변화 강조**(현재 스텝 테두리·포커스 글로우·에러 시 빨간 테두리)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **사이버 미니멀:** 배경 #0f172a, 서피스 #1e293b, Inter, 시안 액센트 #22d3ee, 미니멀 카드.
- **단일 포커스 입력 흐름:** 1단계 이메일 → 2단계 이름 → 3단계 비밀번호 → 4단계 비밀번호 확인·가입하기. 한 화면에 현재 스텝 하나만 표시, 다음/이전으로 포커스 이동.
- **상태 변화 강조:** `.s93-step--active`로 현재 스텝 카드에 시안 테두리·글로우, 입력 포커스 시 시안 박스섀도우, `.s93-field--invalid` 시 빨간 테두리·에러 문구 강조.
- **중앙 정렬:** `.signup-fixed` + `.s93-layout`으로 뷰포트 가로·세로 중앙, max-width 400px 반응형.
- **반응형:** clamp·100dvh·단일 컬럼으로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 단계별 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh로 PC·모바일 전 범위 대응.
