# 시안 103 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 실험적 인터페이스  
- **UI 구조:** 단계 분할형(Progressive Form)  
- **인터랙션:** 입력 반응형 애니메이션  

---

## 콘셉트

다크 톤·퍼플 액센트·Space Grotesk의 **실험적 인터페이스** 무드에, **단계 분할형(Progressive Form)**(01 기본 → 02 보안, 단계 인디케이터·커넥터)과 **입력 반응형 애니메이션**(포커스 시 라벨·테두리·2px 링 전환)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **실험적 인터페이스:** 배경 #0c0c0e, 서피스 #18181b, Space Grotesk, 퍼플 액센트 #a78bfa, 단계 번호 01/02·커넥터 라인, 다크 테마.
- **단계 분할형(Progressive Form):** 상단 단계 인디케이터(01 기본 / 02 보안), 1단계 패널(이메일·이름·다음), 2단계 패널(비밀번호·비밀번호 확인·이전·가입하기). 현재 단계만 표시, 다음/이전 시 포커스 이동.
- **입력 반응형 애니메이션:** `.s103-field--focus`로 포커스 시 라벨 색·테두리·2px 링 퍼플 전환, blur 시 원복, 에러 시 빨간 테두리·라벨 강조.
- **중앙 정렬:** `.signup-fixed` + `.s103-layout`으로 뷰포트 가로·세로 중앙, max-width 400px 반응형.
- **반응형:** clamp·100dvh·단일 컬럼으로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 단계별 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·단일 컬럼으로 PC·모바일 전 범위 대응.
