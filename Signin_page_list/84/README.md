# 시안 84 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 네오 브루탈리즘  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** 상태 변화 강조  

---

## 콘셉트

흑백 대비·두꺼운 테두리·각진 섀도우의 **네오 브루탈리즘** 무드에, **카드 분리형 레이아웃**(필드마다 하나의 카드로 시각적 분리)과 **상태 변화 강조**(포커스 시 핑크 테두리/섀도우, 에러 시 빨간 테두리/섀도우, 유효 시 녹색 체크)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **네오 브루탈리즘:** 배경 #f5f5f0, 카드 흰색·3px 검정 테두리·4px 오프셋 박스섀도우, Space Grotesk·IBM Plex Mono, 라운드 없음.
- **카드 분리형 레이아웃:** 이메일·이름·비밀번호·비밀번호 확인 각각 `.s84-card` 하나로 감싸서 한 필드 = 한 카드. 스캔 시 블록 단위로 구분됨.
- **상태 변화 강조:** 포커스 시 카드 `border`·`box-shadow`를 #ff3366으로 전환. invalid 시 #dc2626 테두리/섀도우 + 에러 문구. 유효 입력 시 카드 우측에 녹색 ✓ 표시.
- **중앙 정렬:** `.signup-fixed` + `.s84-wrap`으로 뷰포트 가로·세로 중앙, max-width 420px 반응형.
- **GSAP:** 로드 시 `.s84-wrap` 페이드인 + 위로 이동.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.
