# 시안 130 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 네오 브루탈리즘  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** 포커스 이동 중심 UX  

---

## 콘셉트

베이지 배경(#f5f0e8)·검정 테두리·Noto Sans KR 볼드 기반의 **네오 브루탈리즘** 무드에, **카드 분리형 레이아웃**(필드별로 독립 카드, 3px 테두리·4px 오프셋 그림자)과 **포커스 이동 중심 UX**(Enter로 다음 필드로 포커스 이동, 포커스 시 카드 테두리·그림자 강조)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **네오 브루탈리즘:** 배경 #f5f0e8, 카드 #fff, 3px 검정 테두리, 4px 4px 0 그림자, 라운드 없음, 굵은 폰트.
- **카드 분리형:** 이메일·이름·비밀번호·비밀번호 확인 각각 별도 카드, 포커스/유효/에러 시 테두리·그림자 색 변경.
- **포커스 이동 중심:** Enter로 다음 입력으로 포커스 이동, 포커스 카드 시 시각적 강조(transform·shadow).
- **중앙 정렬:** 가로·세로 중앙, max-width 400px.
- **반응형:** clamp·100dvh로 PC~모바일 전 범위 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·max-width 400px로 PC·모바일 전 범위 대응.
