# 시안 98 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** 입력 반응형 애니메이션  

---

## 콘셉트

따뜻한 크림·블러시 톤의 **감성 테크** 무드에, **카드 분리형 레이아웃**(기본 정보 카드 / 계정 보안 카드)과 **입력 반응형 애니메이션**(포커스 시 라벨·테두리·박스섀도우 전환)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **감성 테크:** 배경 #fdf8f6, Noto Sans KR, 테라코타 액센트 #c97b63, 부드러운 카드 그림자, 좌측 카드 테라코타·우측 카드 퍼플 좌측 보더.
- **카드 분리형 레이아웃:** PC에서 2열 그리드 — 좌측 카드「기본 정보」(이메일·이름), 우측 카드「계정 보안」(비밀번호·비밀번호 확인·가입하기). 620px 이하에서 1열.
- **입력 반응형 애니메이션:** `.s98-field--focus`로 포커스 시 라벨 색·테두리·2px 링 테라코타 전환, blur 시 원복, 에러 시 빨간 테두리·라벨 강조.
- **중앙 정렬:** `.signup-fixed` + `.s98-layout`으로 뷰포트 가로·세로 중앙, max-width 700px(레이아웃)·660px(카드 그리드) 반응형.
- **반응형:** clamp·100dvh·620px 브레이크포인트로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.
