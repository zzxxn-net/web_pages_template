# 시안 129 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 실험적 인터페이스  
- **UI 구조:** 단일 포커스 입력 흐름  
- **인터랙션:** 스크롤 없는 고정 플로우  

---

## 콘셉트

다크 톤(#0f0f12)·민트 악센트(#00d4aa)·JetBrains Mono + Noto Sans KR 기반의 **실험적 인터페이스** 무드에, **단일 포커스 입력 흐름**(한 화면에 한 필드만 노출, 이메일 → 이름 → 비밀번호 → 비밀번호 확인 순으로 다음/이전 이동)과 **스크롤 없는 고정 플로우**(100dvh 고정, overflow hidden, 상단 진행 바로 단계 시각화)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **실험적 인터페이스:** 다크 배경 #0f0f12, 서피스 #1a1a1f, 모노스페이스 타이포·민트 #00d4aa·에러 #ff5c7a, 8px 라운드.
- **단일 포커스 입력 흐름:** 한 번에 하나의 입력만 표시, 다음/이전으로 단계 이동, 단계 라벨(이메일·이름·비밀번호·비밀번호 확인) 표시.
- **스크롤 없는 고정 플로우:** 뷰포트 100dvh 고정, 스크롤 없음, 상단 고정 진행 바(4단계 비율), GSAP으로 슬롯 전환.
- **중앙 정렬:** 가로·세로 중앙, max-width 360px.
- **반응형:** clamp·100dvh로 PC~모바일 전 범위 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·max-width 360px·overflow hidden으로 스크롤 없이 PC·모바일 전 범위 대응.
