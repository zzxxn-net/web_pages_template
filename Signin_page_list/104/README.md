# 시안 104 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 현대 판타지  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** 상태 변화 강조  

---

## 콘셉트

따뜻한 크림·앰버 톤의 **현대 판타지** 무드에, **비대칭 레이아웃**(좌측 브랜드 영역 1fr / 우측 폼 영역 1.4fr)과 **상태 변화 강조**(touched·valid 시 녹색 테두리·배경, invalid 시 빨간 테두리·배경·라벨)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **현대 판타지:** 배경 #faf6f2, 좌측 #f0e9e0, Lora 이탤릭 타이틀·Source Sans 3 본문, 앰버 액센트 #b8860b.
- **비대칭 레이아웃:** grid 1fr 1.4fr — 좌측 aside(회원가입·환상의 세계로 초대합니다), 우측 form-wrap(이메일·이름·비밀번호·비밀번호 확인·가입하기). 600px 이하에서 1열.
- **상태 변화 강조:** `.s104-field--valid`(녹색 #2d6a4f 테두리·배경), `.s104-field--invalid`(빨간 #c41e3a 테두리·배경·라벨), 제출 중 `.s104-btn--busy`로 버튼 비활성·시각 피드백.
- **중앙 정렬:** `.signup-fixed` + `.s104-layout`으로 뷰포트 가로·세로 중앙, max-width 720px·600px 브레이크포인트 반응형.
- **반응형:** clamp·100dvh·600px 브레이크포인트로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh·600px 브레이크포인트로 PC·모바일 전 범위 대응.
