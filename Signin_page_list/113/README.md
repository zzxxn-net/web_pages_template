# 시안 113 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** GSAP 기반 부드러운 전환  

---

## 콘셉트

따뜻한 크림·테라코타 톤과 Noto Sans KR·Outfit 기반의 **감성 테크** 무드에, **비대칭 레이아웃**(PC에서 좌측 브랜드 영역 42%·우측 폼 58%, 모바일에서는 상하 배치)과 **GSAP 기반 부드러운 전환**(페이지 로드 시 전체 페이드인, 좌측 aside·우측 폼 섹션 각각 x 방향 슬라이드 인)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **감성 테크:** 배경 #faf8f6, 좌측 패널 그라데이션(#f5f0eb→#ebe4dc), 액센트 #c17f59, Noto Sans KR·Outfit 본문.
- **비대칭 레이아웃:** 768px 이상에서 grid 42% / 58% 2열(좌: 브랜드·타이틀·태그라인, 우: 폼); 좌측 justify-self: end, 우측 justify-self: start로 시각적 균형.
- **GSAP 부드러운 전환:** inner opacity 0→1, aside x: -20→0, formSection x: 20→0으로 좌·우에서 부드럽게 등장.
- **중앙 정렬:** 뷰포트 가로·세로 중앙, 모바일에서는 세로 스택 후 폼 중앙.
- **반응형:** clamp·100dvh·768px 브레이크포인트로 PC·모바일 전 범위 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- 768px 미만: 1열 세로 배치(브랜드 → 폼). 768px 이상: 2열 비대칭 그리드.
