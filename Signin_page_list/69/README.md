# 시안 69 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 미래적 클린
- **UI 구조:** 비대칭 레이아웃
- **인터랙션:** GSAP 기반 부드러운 전환

---

## 콘셉트

미래적 클린 톤과 비대칭 레이아웃을 결합한 회원가입 MVP 시안입니다. 왼쪽에 타이틀·설명, 오른쪽에 폼 카드를 두어 시선 흐름을 분리하고, GSAP으로 등장·포커스 전환을 부드럽게 연출합니다.

## UI/UX 특징

- **미래적 클린:** 밝은 배경(#f8fafc), DM Sans 타이포, 단일 액센트(#0ea5e9), 여백 중심 레이아웃
- **비대칭 레이아웃:** PC에서 좌(타이틀·설명) / 우(폼 카드) 1fr : 1.1fr 그리드, 모바일에서 세로 스택·중앙 정렬
- **GSAP 전환:** 레이아웃·aside·formWrap 순차 등장(from opacity/x), 입력 포커스 시 scale 1.01

## 사용 의도 및 구조

- **index.html:** 메인 마크업, Alpine.js `signupForm()` 바인딩, HTMX 속성
- **style.scss / style.css:** 시안 전용 스타일, 반응형(639px 브레이크포인트)
- **js/main.ts, js/main.js:** 검증·제출·GSAP 등장/포커스 애니메이션
- PC~모바일 전 범위 중앙 정렬·반응형 대응
