# 시안 102 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 사이버 미니멀  
- **UI 구조:** 카드 분리형 레이아웃  
- **인터랙션:** 포커스 이동 중심 UX  

---

## 콘셉트

그레이·화이트·시안 액센트의 **사이버 미니멀** 무드에, **카드 분리형 레이아웃**(기본 정보 카드 / 계정 보안 카드 2열)과 **포커스 이동 중심 UX**(다음·계정 보안 클릭 시 해당 카드로 스크롤·비밀번호 필드 포커스, 이전·기본 정보 클릭 시 기본 정보 카드·이메일 포커스)를 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **사이버 미니멀:** 배경 #f4f4f5, 카드 #ffffff·Inter, 시안 액센트 #06b6d4, 카드 상단 3px 시안 라인, 포커스 시 2px 링·라벨 색 전환.
- **카드 분리형 레이아웃:** PC에서 2열 그리드 — 좌측 카드「기본 정보」(이메일·이름·다음·계정 보안), 우측 카드「계정 보안」(비밀번호·비밀번호 확인·이전·기본 정보·가입하기). 620px 이하에서 1열.
- **포커스 이동 중심 UX:** 「다음 · 계정 보안」 클릭 시 기본 정보 유효성 검사 후 계정 보안 카드로 smooth 스크롤·비밀번호 필드 포커스; 「이전 · 기본 정보」 클릭 시 기본 정보 카드로 스크롤·이메일 필드 포커스.
- **중앙 정렬:** `.signup-fixed` + `.s102-layout`으로 뷰포트 가로·세로 중앙, max-width 680px(레이아웃)·660px(카드 그리드) 반응형.
- **반응형:** clamp·100dvh·620px 브레이크포인트로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형
