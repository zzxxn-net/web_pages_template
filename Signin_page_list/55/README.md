# 회원가입 페이지 템플릿 시안 55

실험적 인터페이스 톤의 마이크로 인터랙션 중심 회원가입 페이지 MVP 시안입니다.  
스크롤 없는 고정 플로우로 한 화면에서 가입을 완료하고, 필드·버튼에 마이크로 인터랙션을 적용했습니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 실험적 인터페이스  
- **UI 구조:** 마이크로 인터랙션 중심  
- **인터랙션:** 스크롤 없는 고정 플로우  

## 콘셉트 및 UI/UX 특징

- **실험적 인터페이스:** 다크 배경·블러 오브 배경, Space Grotesk 폰트, 보라·인디고 그라데이션 포인트(#a855f7, #6366f1)로 실험적인 톤을 줍니다. 블롭형 카드(둥근 모서리 32px)와 글래스모피즘 스타일을 사용했습니다.
- **마이크로 인터랙션 중심:** 필드 호버 시 미세 이동(translateX), 포커스 시 라벨 색상·입력창 scale·그림자 강조, 비밀번호 보기 버튼 호버 scale, 제출 버튼 호버 시 상승·scale·그림자 변화, 블롭 카드 호버 시 테두리 강조 등 작은 반응을 여러 요소에 적용했습니다.
- **스크롤 없는 고정 플로우:** `overflow: hidden`·`100dvh`로 뷰포트 고정, 필수 입력만 포함해 한 화면에서 가입이 완료되도록 구성했습니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(가입 폼 검증·제출·포커스 상태)  
- **동작:** Alpine.js로 검증·focusedField·제출, GSAP으로 `.s55-container` 등장(scale·opacity) 애니메이션.

## 중앙 정렬 및 반응형

- 모든 해상도에서 `.s55-container`를 가로·세로 중앙에 배치하고, 블롭 카드와 폼은 중앙 정렬을 유지합니다.  
- `max-width: 400px`·패딩으로 PC~모바일 전 범위 반응형을 적용했습니다.
