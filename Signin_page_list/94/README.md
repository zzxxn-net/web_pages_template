# 시안 94 · 회원가입 페이지 템플릿

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 미래적 클린  
- **UI 구조:** 마이크로 인터랙션 중심  
- **인터랙션:** 입력 반응형 애니메이션  

---

## 콘셉트

라이트 그레이·화이트·블루 액센트의 **미래적 클린** 무드에, **마이크로 인터랙션 중심**(버튼 호버·액티브 스케일·포커스 링)과 **입력 반응형 애니메이션**(포커스 시 라벨·테두리·박스섀도우 전환)을 결합한 회원가입 MVP 시안입니다.

## UI/UX 특징

- **미래적 클린:** 배경 #f8fafc, 카드 #ffffff, DM Sans, 블루 액센트 #3b82f6, 미니멀 그림자.
- **마이크로 인터랙션 중심:** 버튼 호버 시 translateY(-1px)·박스섀도우, 액티브 시 scale(0.98), 비밀번호 보기 버튼 액티브 scale(0.96).
- **입력 반응형 애니메이션:** `.s94-field--focus`로 포커스 시 라벨 색·테두리·3px 링 전환, blur 시 원복, 에러 시 빨간 테두리·라벨 강조.
- **중앙 정렬:** `.signup-fixed` + `.s94-layout`으로 뷰포트 가로·세로 중앙, max-width 400px 반응형.
- **반응형:** clamp·100dvh·단일 컬럼으로 PC·모바일 대응.

## 사용 의도 및 구조

- **MVP 회원가입:** 이메일, 이름, 비밀번호, 비밀번호 확인 + 유효성 검사 + 제출. HTMX + Alpine.js + TypeScript + SASS + GSAP.
- **파일:** index.html, style.scss → style.css, js/main.ts → main.js, README.md.

## 반응형

- clamp·100dvh로 PC·모바일 전 범위 대응.
