# 회원가입 페이지 템플릿 시안 61

사이버 미니멀 톤의 비대칭 레이아웃 회원가입 페이지 MVP 시안입니다.  
좌측에 헤더·포커스 힌트, 우측에 폼을 두고 포커스 이동 시 현재 입력 필드를 강조합니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 사이버 미니멀  
- **UI 구조:** 비대칭 레이아웃  
- **인터랙션:** 포커스 이동 중심 UX  

## 콘셉트 및 UI/UX 특징

- **사이버 미니멀:** 다크 배경(#0f1419)·Space Grotesk·JetBrains Mono, 시안 액센트(#58a6ff)로 테크한 미니멀 톤을 줍니다.
- **비대칭 레이아웃:** 그리드 1fr / 1.4fr로 좌측에 타이틀·설명·포커스 힌트, 우측에 입력 폼을 배치해 시선 흐름을 한 방향으로 유지합니다. 모바일에서는 세로 쌓기로 전환합니다.
- **포커스 이동 중심 UX:** 포커스된 필드에 따라 좌측 aside에 현재 입력 항목명(이메일·이름·비밀번호·비밀번호 확인)을 표시하고, 해당 필드에 테두리·라벨 색상을 적용해 포커스 이동을 시각적으로 강조합니다. GSAP으로 레이아웃 등장 시 좌/우 블록을 분리 등장시킵니다.

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP  
- **파일:** `index.html`(마크업), `style.scss` → `style.css`(시안 전용 스타일), `js/main.ts` → `js/main.js`(검증·제출·focusLabel·GSAP 등장)  
- **동작:** Alpine.js로 focusedField·focusLabel·검증·제출, GSAP으로 `.s61-layout`·`.s61-aside`·`.s61-form-wrap` 등장 애니메이션.

## 중앙 정렬 및 반응형

- 전체 `.s61-layout`은 뷰포트 가로·세로 중앙에 배치하고, 내부는 비대칭 그리드로 좌/우 비율을 다르게 유지합니다.  
- `max-width: 720px`·그리드·미디어쿼리(600px 이하 1열)로 PC~모바일 전 범위 반응형을 적용했습니다.
