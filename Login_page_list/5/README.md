# Login Page — Design 5

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 감성 테크

UI 구조: 마이크로 인터랙션 중심

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

부드러운 그라데이션 배경과 라벤더 톤 악센트로 감성 테크 무드를 주고, 포커스·호버·클릭마다 작은 반응(라벨 색상·필드 스케일·버튼 떠오름)을 넣은 MVP 로그인 페이지입니다.  
한 화면 내에서 스크롤 없이 고정 플로우로 완결됩니다.

## UI/UX 특징

- **감성 테크:** 연한 보라·하늘 그라데이션(#f0ecf9~#e8f4fc), 반투명 카드·블러, 보라 악센트(#8b7cf0)
- **마이크로 인터랙션 중심:** 포커스 시 필드 스케일(1.01)·라벨 색상 전환, 버튼 호버 시 translateY(-1px)·섀도우, 토글 버튼 active scale(0.97)
- **스크롤 없는 고정 플로우:** fixed 중앙 정렬, 한 화면 내에서 폼 완결, 스크롤 최소화
- **중앙 정렬·반응형:** PC·모바일 모두 가로·세로 중앙 정렬, 480px 이하에서 패딩 조정
- **접근성:** 라벨·placeholder·버튼 비활성/로딩·aria 반영

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(폼 상태·is-focused 토글), GSAP(카드 진입), SCSS(감성 톤·마이크로 인터랙션 스타일)
- **접근성:** focus-visible·라벨·placeholder 반영
