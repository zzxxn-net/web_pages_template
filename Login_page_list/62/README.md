# 로그인 페이지 템플릿 시안 62

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 네오 브루탈리즘
- **UI 구조:** 단일 포커스 입력 흐름
- **인터랙션:** GSAP 기반 부드러운 전환

## 콘셉트 및 UI/UX 특징

- **네오 브루탈리즘:** 굵은 검정 테두리(3px), 오프셋 박스 섀도우(10px 10px 0 #1a1a1a), 원색 대비(흰 카드·검정 버튼)로 브루탈한 톤을 적용했습니다.
- **단일 포커스 입력 흐름:** 포커스 중인 필드만 `is-active`(풀 opacity·스케일·강한 섀도우), 나머지는 `is-dimmed`(낮은 opacity·약한 스케일)로 한 번에 하나의 입력에만 시각적 집중을 두었습니다.
- **GSAP 기반 부드러운 전환:** 카드 진입 시 페이드·슬라이드, 필드 포커스/블러 시 스케일 전환으로 부드럽게 반응합니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 네오 브루탈 카드, 단일 포커스용 `activeField`·`is-active`/`is-dimmed` 클래스 마크업.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(네오 브루탈 톤, 단일 포커스 시 상태 스타일).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`neoBrutalSingleFocusLogin`), 폼 제출, GSAP 카드 진입·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
