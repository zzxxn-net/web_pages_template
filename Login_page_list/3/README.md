# Login Page — Design 3

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 미래적 클린

UI 구조: 비대칭 레이아웃

인터랙션: 포커스 이동 중심 UX

## 콘셉트

밝은 배경과 클린한 타이포그래피를 바탕으로, 왼쪽 비주얼 패널과 오른쪽 폼 패널을 나누는 비대칭 레이아웃의 MVP 로그인 페이지입니다.  
포커스 링·tab 순서·focus-visible을 활용해 키보드/포커스 이동 중심 UX를 강조했습니다.

## UI/UX 특징

- **비대칭 레이아웃:** PC(900px 이상)에서 왼쪽 42% 비주얼(그라데이션·도형), 오른쪽 58% 폼. 모바일에서는 상단 비주얼·하단 폼으로 세로 배치
- **미래적 클린:** 밝은 배경(#f8fafc), 화이트 폼 패널, 블루·인디고 그라데이션 악센트(#0ea5e9~#8b5cf6)
- **포커스 이동 중심 UX:** 입력·버튼·링크에 focus-visible 링 적용, 포커스 시 그룹(.is-focused) 스타일·GSAP 연동
- **중앙 정렬·반응형:** 모든 디바이스에서 가로·세로 중앙 정렬, 768px·900px 브레이크포인트로 패딩·레이아웃 전환
- **스크롤 없는 고정 플로우:** 한 화면 내에서 폼 완결

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(폼 상태·포커스 그룹), GSAP(비주얼·폼 패널 진입 애니메이션), SCSS(비대칭·포커스 링)
- **접근성:** focus-visible, tabindex, aria-busy·라벨·placeholder 반영
