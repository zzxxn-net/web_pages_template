# 로그인 페이지 템플릿 시안 59

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** 입력 반응형 애니메이션

## 콘셉트 및 UI/UX 특징

- **감성 테크:** 따뜻한 베이지·라벤더 그라데이션 배경(#f5e6e8 → #e8e4f0)과 보라 톤 포인트(#8b6b8c)로 감성적이면서도 테크한 분위기를 적용했습니다.
- **카드 분리형 레이아웃:** 브랜딩/환영 문구용 카드와 로그인 폼용 카드를 시각적으로 분리했습니다. PC에서는 가로로 두 카드(42% : 58%) 배치, 모바일에서는 세로로 쌓습니다.
- **입력 반응형 애니메이션:** GSAP으로 카드 진입 시 스태거 페이드·슬라이드, 입력 필드 포커스 시 스케일·포커스 링·배경 변화 등 입력에 반응하는 애니메이션을 적용했습니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 카드 분리형 마크업(브랜드 카드 + 폼 카드).
- `style.scss` / `style.css`: 해당 시안 전용 스타일(감성 테크 톤, 반응형 카드 분리).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`senseTechSplitLogin`), 폼 제출 처리, GSAP 카드 스태거·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
