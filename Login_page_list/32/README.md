# 로그인 페이지 템플릿 시안 32

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 다크 아카이브
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** 포커스 이동 중심 UX

---

## 콘셉트 및 UI/UX 특징

- **다크 아카이브:** 딥 그레이 배경(#1a1918), 카드(#242220), 앰버 골드 악센트(#c9a227)로 아카이브/도서관 느낌의 어두운 무드를 적용했습니다. 브랜딩 카드 하단 그라데이션 바로 포인트를 줍니다.
- **카드 분리형 레이아웃:** 데스크톱에서는 좌측(브랜딩·타이틀·문구) / 우측(로그인 폼) 두 카드를 나란히 배치하고, 모바일에서는 상·하로 쌓아 동일한 분리 구조를 유지합니다. 가로·세로 중앙 정렬을 기본으로 합니다.
- **포커스 이동 중심 UX:** 이메일 → 비밀번호 → 버튼 순의 탭 순서, `focus-visible` 스타일, Alpine으로 현재 포커스 필드에 `has-focus` 클래스 적용해 테두리·글로우 강조. GSAP으로 포커스 시 필드 scale 1.01, 블러 시 1로 복귀하는 마이크로 인터랙션을 적용했습니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 버튼만 제공합니다.
- PC~모바일 반응형: 640px 이하에서 카드가 세로 스택되며, 전체는 중앙 정렬로 일관됩니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리 시 두 카드가 좌·우에서 슬라이드 인, 포커스 이동 시 필드 scale 애니메이션으로 포커스 중심 UX를 강조합니다.
