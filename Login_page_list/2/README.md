# Login Page — Design 2

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 다크 아카이브

UI 구조: 풀스크린 몰입형

인터랙션: 입력 반응형 애니메이션

## 콘셉트

카드 없이 전체 화면을 쓰는 몰입형 레이아웃에, 아카이브/도서관 느낌의 다크 톤을 적용한 MVP 로그인 페이지입니다.  
배경 그라데이션과 미세한 노이즈로 깊이감을 주고, 입력 포커스 시 라벨·밑줄이 반응하는 인터랙션으로 집중도를 높였습니다.

## UI/UX 특징

- **풀스크린 몰입형:** 카드 없이 화면 전체를 배경으로 사용, 콘텐츠만 중앙에 배치
- **다크 아카이브:** 따뜻한 다크(#0f0c08, #1a1510)와 세피아 악센트(#c4a574), 세리프 타이틀으로 아카이브 무드
- **입력 반응형 애니메이션:** 포커스/값 유무에 따라 라벨 이동·스케일, 밑줄 scaleX·색상 전환, GSAP로 밑줄 애니메이션
- **중앙 정렬·반응형:** PC·모바일 모두 가로·세로 중앙 정렬, 480px 이하에서 패딩 조정
- **스크롤 없는 고정 플로우:** 한 화면 내에서 폼 완결, 스크롤 최소화

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(폼 상태·포커스/블러), GSAP(밑줄 scaleX·진입 애니메이션), SCSS(풀스크린·필드 상태)
- **접근성:** 라벨·placeholder·버튼 비활성/로딩·aria 속성 반영
