# 로그인 페이지 템플릿 시안 60

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 현대 판타지
- **UI 구조:** 단계 분할형(Progressive Form)
- **인터랙션:** 상태 변화 강조

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 다크 그린·차콜 배경(#1a1f1e)과 골드 포인트(#c4a84a), Georgia·Noto Serif 톤으로 판타지 분위기를 냅니다.
- **단계 분할형(Progressive Form):** 1단계(이메일) → 2단계(비밀번호)로 나누고, 상단 인디케이터(1–2 도트·연결선)와 패널 전환으로 진행 상황을 표시합니다.
- **상태 변화 강조:** 단계·포커스·값 유무·로딩에 따라 인디케이터(활성/완료), 필드 테두리·라벨 색, 다음 버튼(is-ready), 접속 버튼(is-loading) 등 시각적 변화를 뚜렷하게 적용했습니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 단계 인디케이터·패널 1/2 마크업.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(현대 판타지 톤, 단계·상태별 스타일).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`modernFantasyProgLogin`), `goStep2`, 폼 제출, GSAP 박스 진입·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
