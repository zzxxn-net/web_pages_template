# Login Page — Design 8

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 다크 아카이브

UI 구조: 비대칭 레이아웃

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

따뜻한 다크 배경과 세피아 악센트로 다크 아카이브 무드를 주고, 좌측 좁은 스트립(세로 텍스트 ARCHIVE)과 우측 폼 영역을 나누는 비대칭 레이아웃의 MVP 로그인 페이지입니다.  
한 화면 내에서 스크롤 없이 고정 플로우로 완결됩니다.

## UI/UX 특징

- **다크 아카이브:** 배경(#0f0c08, #1a1510), 세피아 악센트(#c4a574), 세리프 타이틀·아카이브 톤
- **비대칭 레이아웃:** PC(900px 이상)에서 좌측 28% 스트립(세로 "ARCHIVE"), 우측 72% 폼. 모바일에서는 상단 스트립·하단 폼
- **스크롤 없는 고정 플로우:** fixed 중앙 정렬, 한 화면 내에서 폼 완결, 스크롤 최소화
- **중앙 정렬·반응형:** 모든 디바이스에서 가로·세로 중앙 정렬, 768px·900px 브레이크포인트
- **GSAP:** 스트립·카드 진입 애니메이션

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(폼 상태), GSAP(진입 애니메이션), SCSS(비대칭·다크 아카이브 톤)
- **접근성:** 라벨·placeholder·버튼 비활성/로딩·aria 반영
