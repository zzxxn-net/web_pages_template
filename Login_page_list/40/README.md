적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 실험적 인터페이스
- **UI 구조:** 단일 포커스 입력 흐름
- **인터랙션:** 상태 변화 강조

---

## 콘셉트 및 UI/UX 특징

- **실험적 인터페이스:** 다크 퍼플 배경(#0f0d14), 퍼플 악센트(#b464dc), 카드에 약간의 회전(rotate 0.4deg)을 주어 실험실 느낌의 비정형 레이아웃을 적용했습니다.
- **단일 포커스 입력 흐름:** 한 번에 하나의 필드만 시각적으로 강조합니다. 포커스된 필드는 `is-active`(테두리·글로우·라벨 색·scale 1.02), 나머지는 `is-dimmed`(opacity·scale 감소)로 구분해 현재 입력에만 집중하도록 합니다.
- **상태 변화 강조:** 포커스/블러 시 SCSS 전환과 GSAP scale(1.03 ↔ 1)로 상태 전환을 뚜렷히 하고, 로딩 시 버튼 opacity·shadow 감소로 상태를 명확히 표현합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호·비밀번호 표시 토글·로그인 버튼만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩·카드 회전만 조정합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리 시 카드 회전·포커스 시 필드 scale로 상태 변화를 강조합니다.
