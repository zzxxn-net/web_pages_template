# Login Page — Design 1

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 사이버 미니멀

UI 구조: 카드 분리형 레이아웃

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

다크 배경 위에 단일 카드로 로그인 폼을 분리한, 미니멀한 사이버 톤의 MVP 로그인 페이지입니다.  
불필요한 장식을 제거하고 포커스 입력 흐름과 전환만으로 완결감을 줍니다.

## UI/UX 특징

- **카드 분리형:** 로그인 폼이 한 장의 카드로 분리되어 배경과 명확히 구분됨
- **중앙 정렬:** PC·모바일 모두 가로·세로 중앙 정렬로 일관된 진입점 제공
- **반응형:** 480px 이하에서 패딩·타이포 축소로 가독성 유지
- **포커스/전환:** 포커스 링(accent), GSAP로 카드 등장 애니메이션 적용
- **접근성:** 라벨·placeholder·버튼 비활성/로딩 상태·aria 속성 반영

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할 분담:**  
  - Alpine.js: 비밀번호 표시/숨기기, 로딩 상태, 폼 상태  
  - HTMX: 폼 제출 확장용(현재는 MVP로 클라이언트 처리)  
  - GSAP: 페이지 로드 시 카드 페이드인·슬라이드업  
  - SCSS: 변수·브레이크포인트·재사용 가능한 컴포넌트 단위 스타일

## 폴더 구조


## 실행 방법

`index.html`을 로컬 서버 또는 정적 호스팅으로 열면 됩니다.  
CDN으로 Alpine.js, HTMX, GSAP을 불러오므로 네트워크 연결이 필요합니다.
