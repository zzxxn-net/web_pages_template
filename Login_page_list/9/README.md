# Login Page — Design 9

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 사이버 미니멀

UI 구조: 풀스크린 몰입형

인터랙션: 상태 변화 강조

## 콘셉트

다크 배경과 네온 악센트(#00d4aa)로 사이버 미니멀 무드를 주고, 카드 없이 전체 화면을 쓰는 풀스크린 몰입형 레이아웃 위에, 대기·입력 중·확인 중 상태를 점(dot)과 텍스트로 강조한 MVP 로그인 페이지입니다.

## UI/UX 특징

- **사이버 미니멀:** 다크 배경(#08090c), 민트/시안 악센트(#00d4aa), 미니멀 타이포
- **풀스크린 몰입형:** 카드 없이 뷰포트 전체 사용, 폼만 중앙에 배치 (1번 시안과 무드 같고 레이아웃·인터랙션 다름)
- **상태 변화 강조:** 상단 상태 인디케이터(점 + "대기"|"입력 중"|"확인 중…"), 포커스/로딩 시 점 색·글로우·펄스 애니메이션
- **중앙 정렬·반응형:** PC·모바일 모두 가로·세로 중앙 정렬, 480px 이하 패딩 조정
- **GSAP:** 콘텐츠 블록 진입 애니메이션

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(status, setStatus, statusClass, statusText), GSAP(진입), SCSS(상태 dot·텍스트 스타일)
- **접근성:** aria-live, 라벨·placeholder·버튼 비활성/로딩 반영
