# 로그인 페이지 템플릿 시안 18

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 감성 테크
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** GSAP 기반 부드러운 전환

---

## 콘셉트 및 UI/UX 특징

- **감성 테크:** 보라 계열 그라데이션 배경(#1e1b2e → #2d2640), 라벤더 악센트(#c9b8e8), 부드러운 글래스 필드로 감성적·테크한 톤을 유지합니다.
- **풀스크린 몰입형:** 카드나 패널 없이 뷰포트 전체를 쓰고, 콘텐츠만 중앙에 배치해 스크롤 없는 단일 화면을 구성합니다.
- **GSAP 기반 부드러운 전환:** 배경·내부 블록·필드 순으로 opacity·y·stagger 엔트리 애니메이션을 적용해 부드럽게 등장합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 패딩 조정으로 전 범위 대응합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 전환만 GSAP로 처리합니다.

## 파일 구조

- `index.html` — 풀스크린 로그인 마크업(그라데이션 배경 + 폼)
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인 동작·GSAP 엔트리
- `README.md` — 본 설명
