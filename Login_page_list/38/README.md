# 로그인 페이지 템플릿 시안 38

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 감성 테크
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트 및 UI/UX 특징

- **감성 테크:** 베이지 그라데이션 배경(#f8f4ef → #ebe4d8), 웜 톤 악센트(#a67c52), 부드러운 그림자로 따뜻하고 차분한 무드를 줍니다. 카드 없이 배경과 폼만으로 몰입형 레이아웃을 구성합니다.
- **풀스크린 몰입형:** 100dvh 풀스크린, 그라데이션 배경 위에 중앙에 타이틀·폼만 배치해 스크롤 없이 한 화면에서 로그인 플로우가 완결됩니다.
- **입력 반응형 애니메이션:** 포커스 시 GSAP으로 언더라인 width 0→100%, 필드 scale(1.02). 블러 시 언더라인 0, scale(1) 복귀. CSS focus-within으로도 언더라인 전환을 보조합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호·비밀번호 표시 토글·로그인 버튼만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩만 조정합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리·포커스 시 입력 반응형 GSAP 애니메이션을 적용합니다.

- `js/login.ts` → `js/login.js` — 로그인·GSAP 입력 반응
