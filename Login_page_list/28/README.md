# 로그인 페이지 템플릿 시안 28

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 네오 브루탈리즘
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트 및 UI/UX 특징

- **네오 브루탈리즘:** 밝은 베이지 배경(#f5f3ef), 굵은 검정 테두리(3px), 각진 형태(radius 0), 오프셋 박스 섀도우(6px 6px 0)로 로우·강한 인상을 줍니다. 버튼은 검정 배경·흰 글자·호버 시 살짝 눌리는 느낌(translate + 그림자 축소)으로 표현합니다.
- **카드 분리형 레이아웃:** 브랜드 카드(LOGIN·Access)와 폼 카드를 분리해 세로로 배치하고, 각 카드에 동일한 브루탈 스타일을 적용해 시각적 위계를 둡니다.
- **입력 반응형 애니메이션:** 입력 필드 포커스 시 GSAP으로 scale(1.02), 블러 시 scale(1)로 부드럽게 전환합니다. 엔트리 시 브랜드 카드는 위에서, 폼 카드는 아래에서 슬라이드 인합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 브랜드 카드 + 폼 카드 순서로 일관된 레이아웃을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 애니메이션과 입력 반응형(포커스 scale)을 사용합니다.

## 파일 구조

- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인·GSAP 엔트리·입력 반응형
