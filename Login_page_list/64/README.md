# 로그인 페이지 템플릿 시안 64

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 현대 판타지
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** GSAP 기반 부드러운 전환

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 다크 그린·차콜 배경(#1c211f)과 골드 포인트(#b8a050), Georgia·Noto Serif로 판타지 분위기를 적용했습니다.
- **카드 분리형 레이아웃:** 브랜드/환영용 카드(Realm)와 로그인 폼용 카드를 시각적으로 분리했습니다. PC에서는 가로 40:60 비율, 모바일에서는 세로 스택입니다.
- **GSAP 기반 부드러운 전환:** 카드 진입 시 스태거 페이드·슬라이드, 입력 필드 포커스 시 스케일 전환으로 부드럽게 반응합니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 카드 분리형 마크업(브랜드 카드 + 폼 카드).
- `style.scss` / `style.css`: 해당 시안 전용 스타일(현대 판타지 톤, 반응형 카드 분리).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`fantasyCardSplitLogin`), 폼 제출, GSAP 카드 스태거·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
