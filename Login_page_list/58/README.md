# 로그인 페이지 템플릿 시안 58

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 미래적 클린
- **UI 구조:** 마이크로 인터랙션 중심
- **인터랙션:** 포커스 이동 중심 UX

## 콘셉트 및 UI/UX 특징

- **미래적 클린:** 밝은 배경(#f1f5f9), 흰색 카드, 스카이 블루 포인트(#0ea5e9)로 깔끔하고 미래지향적인 톤을 적용했습니다.
- **마이크로 인터랙션 중심:** 카드 진입 시 GSAP 페이드·슬라이드, 입력 필드 포커스 시 약한 스케일·포커스 링, 버튼 호버/액티브 시 작은 변화로 한 화면 안에서 미세한 피드백을 강조합니다.
- **포커스 이동 중심 UX:** 이메일 입력 후 Enter로 비밀번호 필드로 포커스 이동(`focusPassword`), 포커스/값 유무에 따른 라벨·테두리 색 변화로 입력 순서와 상태를 명확히 합니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 중앙 카드형 마크업, 포커스 이동을 위한 `x-on:keydown.enter.prevent="focusPassword()"` 적용.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(미래적 클린 톤, 마이크로 인터랙션용 transition).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`futureCleanMicroLogin`), `focusPassword`, 폼 제출 처리, GSAP 카드 진입·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
