# 로그인 페이지 템플릿 시안 34

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 실험적 인터페이스
- **UI 구조:** 마이크로 인터랙션 중심
- **인터랙션:** 스크롤 없는 고정 플로우

---

## 콘셉트 및 UI/UX 특징

- **실험적 인터페이스:** 다크 퍼플 배경(#0c0b0f), 핑크 악센트(#ff64b4), 카드에 약간의 회전(rotate -0.5deg)을 주어 정형화되지 않은 실험실 느낌을 줍니다. 타이틀·설명·폼이 한 카드 안에 담깁니다.
- **마이크로 인터랙션 중심:** 포커스 시 입력 필드 하단에 GSAP으로 언더라인 width 0→100% 애니메이션, 버튼 호버 시 scale(1.02)·active 시 scale(0.98), 토글 버튼 호버 시 scale(1.05)·색상 변화. 모든 상호작용에 짧은 피드백을 줍니다.
- **스크롤 없는 고정 플로우:** body·main에 overflow: hidden, min-height: 100dvh로 뷰포트 내에서만 표시되며 스크롤 없이 로그인 플로우가 완결됩니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호·비밀번호 표시 토글·로그인 버튼만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩만 조정합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리 시 카드 scale·회전, 포커스 시 언더라인 GSAP으로 마이크로 인터랙션을 적용합니다.
