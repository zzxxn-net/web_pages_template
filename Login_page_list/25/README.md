# 로그인 페이지 템플릿 시안 25

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 다크 아카이브
- **UI 구조:** 마이크로 인터랙션 중심
- **인터랙션:** 스크롤 없는 고정 플로우

---

## 콘셉트 및 UI/UX 특징

- **다크 아카이브:** 짙은 브라운·세피아 톤(#0d0c0b, #1a1816, #22201d), 베이지 악센트(#b4a896)로 아카이브/도서관 무드를 표현합니다. 카드 상단 "Archive" 배지와 부드러운 테두리로 문서감을 줍니다.
- **마이크로 인터랙션 중심:** 입력 필드 포커스 시 GSAP으로 box-shadow 하이라이트, 로그인 버튼 호버 시 scale(1.02) 적용. 엔트리 시 카드가 살짝 위로 올라오며 등장합니다.
- **스크롤 없는 고정 플로우:** 뷰포트 전체에서 가로·세로 중앙에 카드 하나만 배치해 스크롤 없이 한 화면에서 로그인만 수행합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 중앙 정렬, 480px 이하에서 패딩만 축소해 동일한 고정 플로우를 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 애니메이션과 포커스/호버 마이크로 인터랙션을 사용합니다.

## 파일 구조

- `index.html` — 아카이브 카드·폼 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인·GSAP 마이크로
- `README.md` — 본 설명
