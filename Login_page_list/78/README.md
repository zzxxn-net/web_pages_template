# 로그인 페이지 템플릿 시안 78

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 사이버 미니멀
- **UI 구조:** 단일 포커스 입력 흐름
- **인터랙션:** GSAP 기반 부드러운 전환

## 콘셉트 및 UI/UX 특징

- **사이버 미니멀:** 다크 배경(#0d0d0f), 서페이스(#161618), 시안 악센트(#22d3ee), 모노스페이스(JetBrains Mono) 타이포로 사이버·미니멀 톤을 적용했습니다.
- **단일 포커스 입력 흐름:** 한 화면에 이메일·비밀번호를 모두 노출하되, 하단 보더만으로 구분한 단일 폼으로 배치해 사용자가 위에서 아래로 한 방향으로만 시선과 포커스를 이동하도록 설계했습니다.
- **GSAP 기반 부드러운 전환:** 래퍼·카드 진입 시 opacity + y 트윈으로 부드럽게 등장하고, 포커스 시 필드 상태 전환이 CSS transition과 함께 일관된 이징으로 적용됩니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 단일 카드(cyber-card) 내 이메일·비밀번호 필드, has-focus/has-value/is-loading 클래스.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(사이버 미니멀 톤, 단일 포커스 흐름, 하단 라인 포커스).
- `js/login.ts` → `js/login.js`: Alpine 데이터(cyberSingleFocusLogin), GSAP 래퍼·카드·필드 진입 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
