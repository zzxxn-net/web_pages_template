적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 감성 테크
- **UI 구조:** 단일 포커스 입력 흐름
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트 및 UI/UX 특징

- **감성 테크:** 밝은 베이지 배경(#f8f6f3), 상단 레이디얼 그라데이션, 웜 톤 악센트(#c4a574)로 따뜻하고 차분한 무드를 줍니다. 카드와 필드는 부드러운 테두리·그림자로 감성적인 톤을 유지합니다.
- **단일 포커스 입력 흐름:** 한 번에 하나의 필드만 `is-active`로 강조합니다. 포커스 시 해당 필드만 scale(1.02)·opacity(1)·테두리·그림자로 하이라이트하고, 나머지 필드는 opacity(0.9)·scale(1)로 약하게 표시합니다.
- **입력 반응형 애니메이션:** 포커스/블러 시 GSAP으로 scale·opacity·box-shadow를 부드럽게 전환하고, 카드 엔트리 시 위로 살짝 올라오며 등장합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩만 축소해 동일한 레이아웃을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 단일 포커스 상태와 입력 반응형 애니메이션을 사용합니다.
