# 로그인 페이지 템플릿 시안 30

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 현대 판타지
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** GSAP 기반 부드러운 전환

---

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 짙은 브라운 배경(#1a1612), 골드 악센트(#c9a227), 레이디얼 그라데이션·타이틀 글로우로 판타지 무드를 현대적으로 표현합니다.
- **풀스크린 몰입형:** 뷰포트 전체를 쓰고, 중앙에 제목·설명·폼만 배치해 스크롤 없이 한 화면에서 로그인만 수행합니다.
- **GSAP 기반 부드러운 전환:** 엔트리 시 전체 블록·타이틀·필드·버튼을 순차적으로 opacity·y로 부드럽게 등장시킵니다. 포커스 시 필드 테두리·그림자는 CSS transition으로 처리합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 풀스크린 내 고정 플로우로 동일한 몰입형 레이아웃을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 시퀀스 애니메이션만 사용합니다.

## 파일 구조

- `index.html` — 배경·중앙 블록·폼 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인·GSAP 엔트리
- `README.md` — 본 설명
