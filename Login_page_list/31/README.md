# 로그인 페이지 템플릿 시안 31

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 감성 테크
- **UI 구조:** 단계 분할형(Progressive Form)
- **인터랙션:** GSAP 기반 부드러운 전환

---

## 콘셉트 및 UI/UX 특징

- **감성 테크:** 밝은 베이지 배경(#f5f2ed), 웜 톤 악센트(#b8926a), 부드러운 테두리·그림자로 따뜻하고 차분한 무드를 줍니다. 단계 인디케이터와 버튼에 동일한 톤을 적용합니다.
- **단계 분할형(Progressive Form):** 1단계(이메일 입력 → 다음), 2단계(비밀번호 입력 → 이전/로그인)로 나누고, 단계 인디케이터(1·2·연결선)와 패널 전환을 GSAP으로 부드럽게 처리합니다.
- **GSAP 기반 부드러운 전환:** 엔트리 시 카드·단계 스텝이 opacity·y·scale로 등장하고, 단계 전환 시 패널이 슬라이드(x)·opacity로 전환됩니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일 입력 후 다음 → 비밀번호 입력 후 로그인/이전. 비밀번호 표시 토글만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 카드 하나로 단계 전환만 있어 동일한 레이아웃을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리·단계 전환 GSAP 애니메이션을 사용합니다.

## 파일 구조

- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인·단계·GSAP
- `README.md` — 본 설명
