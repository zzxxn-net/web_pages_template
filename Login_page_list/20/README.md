# 로그인 페이지 템플릿 시안 20

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 네오 브루탈리즘
- **UI 구조:** 마이크로 인터랙션 중심
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트 및 UI/UX 특징

- **네오 브루탈리즘:** 굵은 테두리(3px), 오프화이트 배경(#f5f5f0), 검정 테두리(#1a1a1a), 레드 악센트(#e63946), 박스 섀도우 오프셋으로 로우하고 강한 톤을 유지합니다.
- **마이크로 인터랙션 중심:** 버튼 호버 시 translate·box-shadow 감소(눌리는 느낌), 버튼 active 시 scale(0.98) 또는 추가 translate, 토글 포커스 링, 입력 필드 포커스 시 테두리·그림자·scale(1.02) 등 작은 피드백을 반복적으로 적용합니다.
- **입력 반응형 애니메이션:** 포커스 시 필드 is-focused(테두리·그림자), GSAP로 포커스 인/아웃 시 필드 scale(1.02/1) 전환을 적용합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 패딩 조정으로 전 범위 대응합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 마이크로 인터랙션(호버·액티브·포커스)과 입력 반응형(GSAP scale)만 사용합니다.
