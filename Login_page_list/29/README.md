# 로그인 페이지 템플릿 시안 29

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 미래적 클린
- **UI 구조:** 단계 분할형(Progressive Form)
- **인터랙션:** 상태 변화 강조

---

## 콘셉트 및 UI/UX 특징

- **미래적 클린:** 밝은 블루그레이 배경(#f0f4f8), 화이트 카드, 블루 악센트(#0088cc)로 깔끔하고 미래적인 톤을 줍니다. 부드러운 테두리·그림자로 정돈된 인상입니다.
- **단계 분할형(Progressive Form):** 1단계(이메일 입력 → 다음), 2단계(비밀번호 입력 → 이전/로그인)로 나누고, 단계 인디케이터(1·2·연결선)와 패널 전환을 GSAP으로 부드럽게 처리합니다.
- **상태 변화 강조:** 단계별 `is-active`·`is-done`, 입력 필드 `is-filled`(값 입력 시 테두리·그림자), 포커스 시 `:focus-within` 하이라이트, 다음 버튼 `:disabled`(이메일 비어 있음), 로그인 버튼 `is-loading`으로 상태를 시각적으로 강조합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일 입력 후 다음 → 비밀번호 입력 후 로그인/이전. 비밀번호 표시 토글만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 카드 하나로 단계 전환만 있어 동일한 레이아웃을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리·단계 전환 애니메이션과 상태 클래스를 사용합니다.
