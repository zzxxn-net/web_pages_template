# 로그인 페이지 템플릿 시안 17

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 미래적 클린
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트 및 UI/UX 특징

- **미래적 클린:** 밝은 배경(#f8fafc), 화이트 폼 영역, 블루 악센트(#3b82f6), 깔끔한 타이포그래피로 미래적·클린한 톤을 유지합니다.
- **풀스크린 몰입형:** 카드나 사이드 패널 없이 뷰포트 전체를 쓰고, 콘텐츠만 중앙에 배치해 스크롤 없는 단일 화면을 구성합니다.
- **입력 반응형 애니메이션:** 포커스 시 필드 테두리·그림자·라벨 색상 변화, GSAP로 포커스 인/아웃 시 box-shadow 부드럽게 전환해 입력에 반응하는 연출을 적용합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 패딩 조정으로 전 범위 대응합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 애니메이션과 입력 반응형(포커스) 연출만 사용합니다.

## 파일 구조

- `index.html` — 풀스크린 로그인 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인 동작·포커스 상태·GSAP 엔트리·입력 반응형
- `README.md` — 본 설명
