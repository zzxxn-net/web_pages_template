# Login Page — Design 4

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 네오 브루탈리즘

UI 구조: 단계 분할형(Progressive Form)

인터랙션: 상태 변화 강조

## 콘셉트

굵은 테두리·고대비·장식 최소화의 네오 브루탈리즘 톤에, 이메일 → 비밀번호 순의 단계 분할형 폼을 적용한 MVP 로그인 페이지입니다.  
단계 인디케이터(1 — 2)와 GSAP 단계 전환으로 상태 변화를 명확히 강조합니다.

## UI/UX 특징

- **단계 분할형(Progressive Form):** 1단계에서 이메일만 입력 후 "다음" → 2단계에서 비밀번호 입력 후 "로그인". 한 번에 한 입력에 집중
- **네오 브루탈리즘:** 3px 검정 테두리, 박스 섀도우(8px 8px 0), 노란 악센트(#f7e017), 대문자·굵은 서체, border-radius 0
- **상태 변화 강조:** 단계 점(1, 2)과 연결선으로 진행 상태 표시. 완료 단계는 노란색. GSAP으로 1→2 전환 시 패널 슬라이드
- **중앙 정렬·반응형:** PC·모바일 모두 가로·세로 중앙 정렬, 480px 이하에서 패딩·섀도우 축소
- **스크롤 없는 고정 플로우:** 한 화면 내에서 단계만 전환

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(step, email, password, goStep2, handleSubmit), GSAP(단계 전환·진입 애니메이션), SCSS(브루탈리즘 변수·단계 인디케이터)
- **접근성:** 라벨·placeholder·버튼 비활성·로딩 상태 반영
