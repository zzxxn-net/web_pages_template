# 로그인 페이지 템플릿 시안 36

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 현대 판타지
- **UI 구조:** 단계 분할형(Progressive Form)
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 딥 퍼플 배경(#1a1625), 카드(#252030), 골드 악센트(#d4af37)로 판타지 세계관 톤을 적용했습니다. 부드러운 그림자와 테두리 글로우로 몽환적인 분위기를 줍니다.
- **단계 분할형(Progressive Form):** 1단계(이메일 입력 → 다음), 2단계(비밀번호 입력 → 이전/로그인)로 나누고, 단계 인디케이터(1·2·연결선)와 패널 전환을 GSAP 슬라이드로 처리합니다.
- **입력 반응형 애니메이션:** 포커스 시 필드에 SCSS scale(1.01)·테두리·글로우, GSAP으로 scale(1.02) 보강. 블러 시 scale(1) 복귀. 토글 버튼 호버 시 scale(1.03)로 입력에 반응하는 애니메이션을 적용했습니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일 입력 후 다음 → 비밀번호 입력 후 로그인/이전. 비밀번호 표시 토글만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩만 조정합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리·단계 전환·포커스 시 입력 반응형 애니메이션을 사용합니다.
