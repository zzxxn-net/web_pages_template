# 로그인 페이지 템플릿 시안 43

## 개요

**감성 테크** 무드와 **카드 분리형 레이아웃**을 결합한 로그인 MVP 시안입니다. 브랜드 카드와 폼 카드를 분리해 시각적 위계를 주고, GSAP으로 등장·포커스 전환을 부드럽게 연출합니다.

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** GSAP 기반 부드러운 전환

## UI/UX 특징

- **감성 테크:** 따뜻한 그라데이션 배경(#fef6f2 → #f5e8f0), 브랜드 카드 그라데이션(#e8c4d4 → #d4b8e8), 보라·피치 계열 포인트(#a078b0), 부드러운 라운드(16px)로 감성적 톤 유지.
- **카드 분리형:** 상단(모바일) 또는 좌측(520px 이상)에 브랜드 카드(Welcome / 감성과 기술이 만나는 공간), 우측/하단에 로그인 폼 카드로 역할 분리.
- **GSAP:** 래퍼·브랜드 카드·폼 카드 순차 등장(opacity, y, x), 입력 필드 포커스 시 scale(1.02), 버튼 지연 등장으로 자연스러운 플로우 제공.
- **반응형:** 모든 뷰포트에서 가로·세로 중앙 정렬, 520px 미만에서 세로 스택, 520px 이상에서 가로 2열.

## 파일 구조

- `index.html` — 로그인 마크업(Alpine.js `emotionCardLogin`, HTMX 폼, 카드 분리 구조)
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` / `js/login.js` — 로그인 동작 및 GSAP 초기화
- `README.md` — 본 문서

## 사용 기술

- HTMX, Alpine.js, TypeScript(컴파일 → JS), SASS/SCSS, GSAP
