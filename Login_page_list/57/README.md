# 로그인 페이지 템플릿 시안 57

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 다크 아카이브
- **UI 구조:** 비대칭 레이아웃
- **인터랙션:** GSAP 기반 부드러운 전환

## 콘셉트 및 UI/UX 특징

- **다크 아카이브:** 어두운 배경과 세피아/앰버 포인트 컬러로 기록 보관소·아카이브 분위기를 연출합니다.
- **비대칭 레이아웃:** PC에서는 왼쪽(약 58%) 비주얼 영역과 오른쪽(약 42%) 로그인 폼 영역으로 나누어, 한쪽에 브랜딩·설명, 다른 쪽에 입력 폼을 배치합니다. 모바일에서는 비주얼 상단·폼 하단으로 세로 배치됩니다.
- **GSAP 전환:** 진입 시 비주얼은 왼쪽에서, 폼 영역은 오른쪽에서 페이드·슬라이드로 등장하며, 입력 필드 포커스 시 약간의 스케일 전환으로 부드럽게 반응합니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 비대칭 구조의 마크업(비주얼 블록 + 폼 래퍼·폼 박스).
- `style.scss` / `style.css`: 해당 시안 전용 스타일(다크 톤, 반응형 비대칭 레이아웃).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`darkArchiveAsymLogin`), 폼 제출 처리, GSAP 진입·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
