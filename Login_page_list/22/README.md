# 로그인 페이지 템플릿 시안 22

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 미래적 클린
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** 스크롤 없는 고정 플로우

---

## 콘셉트 및 UI/UX 특징

- **미래적 클린:** 밝은 배경(#f0f4f8), 화이트 카드, 블루 악센트(#2563eb), 깔끔한 타이포그래피로 미래적·클린한 톤을 유지합니다.
- **카드 분리형 레이아웃:** 브랜드 카드(로고·태그라인)와 폼 카드를 분리해, 시각적 위계와 정보 구분을 명확히 합니다.
- **스크롤 없는 고정 플로우:** 뷰포트 전체(min-height: 100dvh)에 콘텐츠를 배치하고 flex 중앙 정렬·overflow: hidden으로 스크롤 없이 고정 플로우를 유지합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 패딩·갭 조정으로 전 범위 대응합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 애니메이션만 GSAP로 처리합니다.

## 파일 구조

- `index.html` — 브랜드 카드 + 폼 카드 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인 동작·GSAP 엔트리
- `README.md` — 본 설명
