# 로그인 페이지 템플릿 시안 35

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 네오 브루탈리즘
- **UI 구조:** 비대칭 레이아웃
- **인터랙션:** 스크롤 없는 고정 플로우

---

## 콘셉트 및 UI/UX 특징

- **네오 브루탈리즘:** 밝은 배경(#f5f5f0), 흰 서페이스, 굵은 검정 테두리(3px), 라운드 없음(0), 강한 박스 섀도우(8px 8px 0)로 raw한 브루탈리즘 톤을 적용했습니다. 타이포는 굵은 sans, 대문자 라벨(EMAIL, PASSWORD, LOG IN)로 일관합니다.
- **비대칭 레이아웃:** 데스크톱에서 좌측(1fr) 브랜딩·타이틀, 우측(1.2fr) 폼 영역으로 그리드를 나누어 비대칭 비율을 줍니다. 모바일에서는 상·하로 스택하며 전체 블록은 중앙 정렬됩니다.
- **스크롤 없는 고정 플로우:** body·main에 overflow: hidden, min-height: 100dvh로 뷰포트 내에서만 표시되며 스크롤 없이 로그인 플로우가 완결됩니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호·비밀번호 표시 토글·로그인 버튼만 제공합니다.
- PC~모바일 반응형: 640px 이하에서 비대칭 그리드가 세로 스택되며, 전체는 중앙 정렬로 유지됩니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리 시 레이아웃 블록이 opacity·y·scale로 등장합니다.

## 파일 구조

- `index.html` — 비대칭 그리드·브랜딩·폼 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인·GSAP 엔트리
- `README.md` — 본 설명
