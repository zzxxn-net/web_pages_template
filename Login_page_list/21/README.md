# 로그인 페이지 템플릿 시안 21

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 다크 아카이브
- **UI 구조:** 단계 분할형(Progressive Form)
- **인터랙션:** 상태 변화 강조

---

## 콘셉트 및 UI/UX 특징

- **다크 아카이브:** 아카이브/도서관 느낌의 짙은 톤(#0f0e0c, #1a1816), 차분한 테두리·배경, 뮤트한 악센트(#a89880)로 정적이고 신뢰감 있는 분위기를 유지합니다.
- **단계 분할형(Progressive Form):** 1단계에서 이메일만 입력 후 «다음»으로 2단계 전환, 2단계에서 비밀번호 입력·이전/로그인 버튼으로 진행합니다.
- **상태 변화 강조:** 스텝 인디케이터(1—2)에 is-active·is-done으로 현재·완료 상태 시각화, 버튼 disabled·is-loading으로 비활성·로딩 상태를 명확히 강조합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일 → 비밀번호 순서의 2단계 플로우, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 패딩 조정으로 전 범위 대응합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 단계 전환(GSAP 패널 opacity·x)과 상태 클래스만 사용합니다.

## 파일 구조

- `index.html` — 단계 인디케이터·2단계 패널 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 단계 전환·제출·GSAP 엔트리
- `README.md` — 본 설명
