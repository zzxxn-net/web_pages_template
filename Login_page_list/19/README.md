# 로그인 페이지 템플릿 시안 19

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 사이버 미니멀
- **UI 구조:** 단계 분할형(Progressive Form)
- **인터랙션:** 입력 반응형 애니메이션

---

## 콘셉트 및 UI/UX 특징

- **사이버 미니멀:** 다크 배경(#0a0a0f), 틸 악센트(#00d4aa), 미니멀 테두리·그림자로 사이버·테크한 톤을 유지합니다.
- **단계 분할형(Progressive Form):** 1단계에서 이메일 입력 후 «다음»으로 2단계 전환, 2단계에서 비밀번호·이전/로그인 버튼으로 진행합니다.
- **입력 반응형 애니메이션:** 포커스 시 필드 테두리·그림자 강조(has-focus), GSAP로 포커스 인/아웃 시 필드 scale(1.01/1) 전환, 단계 전환 시 패널 opacity·x 전환을 적용합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일 → 비밀번호 순서의 2단계 플로우, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 패딩 조정으로 전 범위 대응합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 단계 전환·엔트리·입력 반응형만 사용합니다.

## 파일 구조

- `index.html` — 단계 인디케이터(1—2)·2단계 패널 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 단계 전환·제출·GSAP 엔트리·입력 반응형
- `README.md` — 본 설명
