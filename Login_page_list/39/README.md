# 로그인 페이지 템플릿 시안 39

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 다크 아카이브
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** 상태 변화 강조

---

## 콘셉트 및 UI/UX 특징

- **다크 아카이브:** 딥 그레이 배경(#141210), 반투명 서페이스(rgba(26,24,22,0.85)), 앰버 골드 악센트(#c9a227)로 아카이브/도서관 느낌의 어두운 무드를 적용했습니다.
- **풀스크린 몰입형:** 100dvh 풀스크린, 다크 배경 위에 중앙에 타이틀·폼만 배치해 스크롤 없이 한 화면에서 로그인 플로우가 완결됩니다.
- **상태 변화 강조:** 포커스 시 :focus-within으로 테두리·글로우·라벨 색·scale(1.01) 전환, 값 입력 시 has-value(선택) 스타일, 로딩 시 버튼 is-loading(opacity·shadow 감소)로 상태별 시각적 변화를 뚜렷히 표현합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호·비밀번호 표시 토글·로그인 버튼만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩만 조정합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리 시 배경·내부 블록 opacity·y 등장으로 부드럽게 진입합니다.
