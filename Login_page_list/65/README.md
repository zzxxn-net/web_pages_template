# 로그인 페이지 템플릿 시안 65

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 다크 아카이브
- **UI 구조:** 단계 분할형(Progressive Form)
- **인터랙션:** 입력 반응형 애니메이션

## 콘셉트 및 UI/UX 특징

- **다크 아카이브:** 어두운 배경(#0c0a09)과 세피아/앰버 포인트(#b09068)로 기록 보관소·아카이브 분위기를 적용했습니다.
- **단계 분할형(Progressive Form):** 1단계(이메일) → 2단계(비밀번호)로 나누고, 상단 인디케이터(1–2 도트·연결선)와 패널 전환으로 진행 상황을 표시합니다.
- **입력 반응형 애니메이션:** 포커스/값 유무에 따라 필드 테두리·라벨 색·포커스 링이 전환되고, GSAP으로 필드 포커스 시 스케일 애니메이션이 적용됩니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 단계 인디케이터·패널 1/2 마크업, `has-focus`/`has-value` 클래스.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(다크 아카이브 톤, 단계·입력 반응 스타일).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`darkArchProgInputLogin`), `goStep2`, 폼 제출, GSAP 박스 진입·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
