# 로그인 페이지 템플릿 시안 61

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 사이버 미니멀
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** 스크롤 없는 고정 플로우

## 콘셉트 및 UI/UX 특징

- **사이버 미니멀:** 다크 그레이·차콜 배경(#0c0e10)과 시안 블루 포인트(#4a9eff), 미니멀한 라인·보더로 사이버/테크 분위기를 적용했습니다.
- **풀스크린 몰입형:** `position: fixed; inset: 0`으로 전체 화면을 덮고, 중앙에 로그인 카드만 배치하여 몰입형 한 화면 구성을 적용했습니다.
- **스크롤 없는 고정 플로우:** `html`·`body`·`main`에 `overflow: hidden`을 두어 스크롤 없이 한 화면 안에서 모든 입력·버튼이 보이도록 고정 플로우로 설계했습니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 풀스크린 메인 안 단일 카드(헤더·폼) 마크업.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(사이버 미니멀 톤, 풀스크린·오버플로우 숨김).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`cyberMinimalFullLogin`), 폼 제출 처리, GSAP 카드 진입·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
