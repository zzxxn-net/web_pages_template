# 로그인 페이지 템플릿 시안 14

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 실험적 인터페이스  
- **UI 구조:** 단계 분할형(Progressive Form)  
- **인터랙션:** GSAP 기반 부드러운 전환  

---

## 콘셉트 및 UI/UX 특징

- **실험적 인터페이스:** 다크 톤 배경(#0c0c0e), 티포그래피·번호(01, 02) 기반 스텝 인디케이터, 포인트 컬러(#00d4aa)로 실험적인 톤을 유지합니다.
- **단계 분할형(Progressive Form):** 1단계에서 이메일만 입력 후 «다음 단계»로 2단계 전환, 2단계에서 비밀번호 입력·이전/로그인 버튼으로 진행합니다.
- **GSAP 기반 부드러운 전환:** 진입 시 래퍼·스텝 영역 opacity/y·x 애니메이션, 단계 전환 시 패널 opacity·y·scale 전환으로 부드럽게 전환합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일 → 비밀번호 순서의 2단계 플로우, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 중앙 정렬, 모바일에서 스텝 라벨 숨김 등으로 전 범위 대응합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 단계 전환과 엔트리만 GSAP로 처리합니다.

## 파일 구조

- `index.html` — 로그인 마크업(스텝 인디케이터, 2단계 패널)
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 단계 전환·제출·GSAP 엔트리
- `README.md` — 본 설명
