# 로그인 페이지 템플릿 시안 66

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 감성 테크
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** 상태 변화 강조

## 콘셉트 및 UI/UX 특징

- **감성 테크:** 따뜻한 베이지·피치 그라데이션(#fef7f0 → #f5e6dc)과 앰버 포인트(#b8734a)로 부드럽고 감성적인 톤을 적용했습니다.
- **풀스크린 몰입형:** 뷰포트 전체를 쓰는 고정 레이아웃으로, 스크롤 없이 중앙 카드 하나에 이메일·비밀번호·로그인 버튼만 배치한 몰입형 구조입니다.
- **상태 변화 강조:** 필드 idle → focus → value, 버튼 idle → loading 전환 시 테두리·배경·그림자·색상이 뚜렷하게 바뀌고, GSAP으로 카드 진입·필드 포커스 시 스케일 애니메이션을 적용했습니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 풀스크린 래퍼·단일 카드·폼 마크업, `has-focus`/`has-value`/`is-loading` 클래스.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(감성 테크 톤, 상태별 스타일).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`senseFullStateLogin`), 폼 제출, GSAP 카드 진입·필드 포커스 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
