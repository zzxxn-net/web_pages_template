# 로그인 페이지 템플릿 시안 42

## 개요

네오 브루탈리즘 톤과 **단일 포커스 입력 흐름**을 결합한 로그인 MVP 시안입니다. 한 번에 하나의 필드만 강조하고 나머지는 디밍하여 집중도를 높이며, GSAP으로 등장·포커스 전환을 부드럽게 연출합니다.

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 네오 브루탈리즘
- **UI 구조:** 단일 포커스 입력 흐름
- **인터랙션:** GSAP 기반 부드러운 전환

## UI/UX 특징

- **네오 브루탈리즘:** 두꺼운 검정 테두리(3px), 박스 섀도우 오프셋, 무지배경(#f2f0e8), 굵은 서체(700~800)로 거친 구조감 표현.
- **단일 포커스:** 이메일/비밀번호 중 포커스된 필드만 `is-active`(섀도우 강조), 나머지는 `is-dimmed`(opacity 0.5)로 시각적 위계 유지.
- **GSAP:** 페이지 로드 시 래퍼·카드 등장(opacity, y, scale), 포커스 인/아웃 시 필드 scale(1.02 ↔ 1) 전환으로 자연스러운 피드백 제공.
- **반응형:** PC~모바일 중앙 정렬, 480px 이하에서 패딩·섀도우 축소.

## 파일 구조

- `index.html` — 로그인 마크업 (Alpine.js `brutalSingleLogin`, HTMX 폼)
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` / `js/login.js` — 로그인 동작 및 GSAP 초기화
- `README.md` — 본 문서

## 사용 기술

- HTMX, Alpine.js, TypeScript(컴파일 → JS), SASS/SCSS, GSAP
