- **UI 구조:** 단일 포커스 입력 흐름

---

## 콘셉트 및 UI/UX 특징

- **사이버 미니멀:** 다크 그레이 배경(#0e1116), 서페이스(#161b22), 시안 블루 악센트(#58a6ff)로 미니멀한 라인과 타이포를 적용했습니다. 불필요한 장식을 없애고 테두리·그림자만으로 계층을 나눕니다.
- **단일 포커스 입력 흐름:** 한 번에 하나의 필드만 시각적으로 강조합니다. 포커스된 필드는 `is-active`(테두리·글로우·라벨 색상), 나머지 필드는 `is-dimmed`(opacity·scale 감소)로 구분해 사용자가 현재 입력 필드에만 집중하도록 합니다.
- **상태 변화 강조:** 포커스/블러 시 SCSS 전환과 GSAP scale(1.02 ↔ 1)로 상태 전환을 뚜렷히 하고, 로딩 시 버튼 텍스트 변경·비활성화로 상태를 명확히 표현합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호·비밀번호 표시 토글·로그인 버튼만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩만 조정합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리 시 카드 등장, 포커스 시 필드 scale로 상태 변화를 강조합니다.
