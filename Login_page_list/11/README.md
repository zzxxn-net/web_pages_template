# Login Page — Design 11

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 현대 판타지

UI 구조: 풀스크린 몰입형

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

다크 배경과 보라·인디고 글로우로 현대 판타지 무드를 주고, 카드 없이 전체 화면을 쓰는 풀스크린 몰입형 레이아웃 위에, 한 화면 내에서 스크롤 없이 고정 플로우로 완결되는 MVP 로그인 페이지입니다.

## UI/UX 특징

- **현대 판타지:** 다크 배경(#0c0b12), 블러 글로우 오브(보라·인디고), 세리프 타이틀 "Realm", 보라 악센트(#a78bfa)
- **풀스크린 몰입형:** 카드 없이 뷰포트 전체 사용, 폼만 중앙에 배치 (6번과 무드 같고 UI·인터랙션 다름)
- **스크롤 없는 고정 플로우:** fixed 중앙 정렬, 한 화면 내에서 폼 완결, 스크롤 최소화
- **중앙 정렬·반응형:** PC·모바일 모두 가로·세로 중앙 정렬, 480px 이하 패딩 조정
- **GSAP:** 콘텐츠 블록 진입 애니메이션

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(폼 상태), GSAP(진입), SCSS(풀스크린·판타지 톤)
- **접근성:** 라벨·placeholder·버튼 비활성/로딩·aria 반영
