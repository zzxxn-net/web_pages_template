# 로그인 페이지 템플릿 시안 15

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 현대 판타지
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** 상태 변화 강조

---

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 따뜻한 톤(앰버/골드 #c9a227), 짙은 브라운 배경(#1a1814), 파치먼트 느낌의 카드와 부드러운 글로우로 판타지 무드를 현대적으로 표현합니다.
- **카드 분리형 레이아웃:** 브랜드 카드(로고·태그라인)와 폼 카드를 분리해, 시각적 위계와 정보 구분을 명확히 합니다.
- **상태 변화 강조:** 입력 필드에 `is-filled`(값 입력 시 테두리·그림자), `is-focused`(포커스 시 강조 테두리·배경), 로그인 버튼에 `is-loading`(제출 중 opacity·비활성)을 적용해 상태를 시각적으로 강조합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 세로 중앙 정렬, 브랜드 카드 + 폼 카드 순서로 일관된 레이아웃을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 애니메이션과 상태 클래스만 사용합니다.

## 파일 구조

- `index.html` — 브랜드 카드 + 폼 카드 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인 동작·상태·GSAP 엔트리
- `README.md` — 본 설명
