# 로그인 페이지 템플릿 시안 50

## 개요

**사이버 미니멀** 무드와 **마이크로 인터랙션 중심** UI를 결합한 로그인 MVP 시안입니다. **스크롤 없는 고정 플로우**로 한 화면에서만 로그인이 완결되며, 필드·버튼·토글에 작은 반응(포커스 링, 호버/액티브 스케일, 리플, 토글 아이콘 회전)을 적용했습니다.

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 사이버 미니멀
- **UI 구조:** 마이크로 인터랙션 중심
- **인터랙션:** 스크롤 없는 고정 플로우

## UI/UX 특징

- **사이버 미니멀:** 그레이·블루 계열 배경(#eef1f5), 흰색 카드, 얇은 보더(#e2e8f0), 포인트 #3182ce, 12px 라운드로 미니멀 톤 유지.
- **마이크로 인터랙션:** 필드 hover 시 보더 강조, focus 시 링(box-shadow)·GSAP scale(1.01), active 시 리플 레이어·토글/버튼 scale(0.97~0.99). 비밀번호 토글 아이콘(눈 형태)이 열림/닫힘에 따라 회전.
- **스크롤 없는 고정 플로우:** html·body·메인에 overflow: hidden, min-height: 100dvh로 뷰포트 고정. 스크롤 없이 한 화면에서 로그인 플로우 완결.
- **반응형:** PC~모바일 중앙 정렬, 480px 이하에서 패딩 축소.

## 파일 구조

- `index.html` — 로그인 마크업(Alpine.js `cyberMicroLogin`, HTMX 폼, has-focus/has-value, 리플·토글 아이콘)
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` / `js/login.js` — 로그인·GSAP 입장·마이크로 인터랙션(필드 scale, 버튼 펄스)
- `README.md` — 본 문서

## 사용 기술

- HTMX, Alpine.js, TypeScript(컴파일 → JS), SASS/SCSS, GSAP
