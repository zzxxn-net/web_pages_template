# Login Page — Design 12

하나의 독립적인 로그인 페이지 템플릿 디자인 시안입니다.

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 감성 테크

UI 구조: 비대칭 레이아웃

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

부드러운 그라데이션 비주얼과 라벤더 톤으로 감성 테크 무드를 주고, 좌측 비주얼 패널과 우측 폼 패널을 나누는 비대칭 레이아웃 위에, GSAP으로 비주얼·폼 패널이 부드럽게 등장하는 MVP 로그인 페이지입니다.

## UI/UX 특징

- **감성 테크:** 밝은 배경(#f5f3fa), 라벤더·하늘 그라데이션 비주얼(#e8e4f5~#f0e8f5), 보라 악센트(#8b7cf0)
- **비대칭 레이아웃:** PC(880px 이상)에서 좌측 40% 비주얼("Welcome back"), 우측 60% 폼. 모바일에서는 상단 비주얼·하단 폼
- **GSAP 기반 부드러운 전환:** 비주얼·폼 패널 진입 시 opacity·x 슬라이드 전환 (power2.out)
- **중앙 정렬·반응형:** PC·모바일 모두 가로·세로 중앙 정렬, 768px·880px 브레이크포인트
- **스크롤 없는 고정 플로우:** 한 화면 내에서 폼 완결

## 사용 의도 및 구조

- **기술 스택:** HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP
- **역할:** Alpine.js(폼 상태), GSAP(비주얼·패널 진입), SCSS(비대칭·감성 톤)
- **접근성:** 라벨·placeholder·버튼 비활성/로딩·aria 반영
