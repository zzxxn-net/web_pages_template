# 로그인 페이지 템플릿 시안 27

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 사이버 미니멀
- **UI 구조:** 비대칭 레이아웃
- **인터랙션:** GSAP 기반 부드러운 전환

---

## 콘셉트 및 UI/UX 특징

- **사이버 미니멀:** 다크 배경(#0a0a0f), 미니멀한 서페이스(#12121a), 시안 악센트(#00d4aa)로 사이버/테크 무드를 냅니다. 포커스 시 필드 테두리·그림자만으로 정돈된 인상입니다.
- **비대칭 레이아웃:** 그리드로 좌측 72% 폼 영역, 우측 28% 사이드(브랜드·라인·Access 텍스트)를 구성합니다. 모바일에서는 상단 폼·하단 사이드로 전환되며, 사이드의 라인은 세로로 표시됩니다.
- **GSAP 기반 부드러운 전환:** 엔트리 시 폼 영역은 왼쪽에서, 사이드는 오른쪽에서 슬라이드 인하며, 사이드 내 라인은 scaleX로 등장합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 640px 이하에서 비대칭이 세로 스택으로 전환됩니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 애니메이션만 사용합니다.

## 파일 구조

- `index.html` — 폼 영역·사이드 마크업
- `style.scss` / `style.css` — 시안 전용 스타일
- `js/login.ts` → `js/login.js` — 로그인·GSAP 엔트리
- `README.md` — 본 설명
