# 로그인 페이지 템플릿 시안 23

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 현대 판타지
- **UI 구조:** 비대칭 레이아웃
- **인터랙션:** 상태 변화 강조

---

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 딥 퍼플·보라 계열 배경(#0f0d12, #1a1625), 골드 악센트(#d4a84b) 로고, 라벤더 그라데이션 버튼(#b896f0)으로 판타지 무드를 현대적으로 표현합니다. 좌측 비주얼에 원형 데코와 레이디얼 그라데이션을 적용했습니다.
- **비대칭 레이아웃:** 그리드로 좌측 35% 비주얼 패널, 우측 65% 폼 영역을 구성합니다. 모바일에서는 상단 비주얼·하단 폼으로 전환되어 중앙 정렬을 유지합니다.
- **상태 변화 강조:** 입력 필드에 `is-filled`(값 입력 시 테두리·그림자), `is-focused`(포커스 시 보라 테두리·배경), `is-error`(유효성 실패 시 빨간 테두리), 버튼에 `is-loading`(제출 중 비활성·opacity)을 적용해 상태를 시각적으로 강조합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 클라이언트 유효성(빈 값 시 에러 상태), 로그인 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 비대칭 그리드가 640px 이하에서 세로 스택으로 전환됩니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 엔트리 애니메이션(비주얼·폼·데코 도형)과 상태 클래스를 사용합니다.
