# 로그인 페이지 템플릿 시안 70

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 현대 판타지
- **UI 구조:** 비대칭 레이아웃
- **인터랙션:** 입력 반응형 애니메이션

## 콘셉트 및 UI/UX 특징

- **현대 판타지:** 베이지·세피아 배경(#f8f4ee), 다크 사이드(#2d2420→#1e1814 그라데이션), 골드 포인트(#8b6914)로 판타지 포털·리얼름 분위기를 적용했습니다.
- **비대칭 레이아웃:** 좌측 좁은 사이드(브랜드·타이틀)와 우측 넓은 폼 영역을 비율(약 38:62)로 나누고, 560px 이하에서는 상·하로 쌓이도록 반응형으로 구성했습니다.
- **입력 반응형 애니메이션:** 포커스/값 유무(has-focus, has-value)에 따라 필드 테두리·라벨 색·포커스 링이 전환되고, GSAP으로 래퍼 진입·사이드/폼 영역 스태거(x 방향)·필드 포커스 스케일 애니메이션이 적용됩니다.

## 사용 의도 및 구조

- MVP 로그인(이메일·비밀번호) 전제의 시안으로, HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 스택을 사용합니다.
- `index.html`: 비대칭 래퍼·사이드·폼 영역 마크업, `has-focus`/`has-value`/`is-loading` 클래스.
- `style.scss` / `style.css`: 해당 시안 전용 스타일(현대 판타지 톤, 비대칭·입력 반응 스타일).
- `js/login.ts` → `js/login.js`: Alpine 데이터(`fantasyAsymInputLogin`), 폼 제출, GSAP 래퍼·사이드/폼·필드 애니메이션.

PC~모바일 전 범위 반응형이며, 가로·세로 중앙 정렬을 기본 레이아웃 규칙으로 적용합니다.
