# 로그인 페이지 템플릿 시안 37

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 미래적 클린
- **UI 구조:** 단일 포커스 입력 흐름
- **인터랙션:** GSAP 기반 부드러운 전환

---

## 콘셉트 및 UI/UX 특징

- **미래적 클린:** 밝은 배경(#f0f4f8), 화이트 카드, 클린 블루 그레이 악센트(#2d5a87)로 미니멀하고 깔끔한 미래감을 줍니다. 부드러운 테두리·그림자만으로 계층을 나눕니다.
- **단일 포커스 입력 흐름:** 한 번에 하나의 필드만 시각적으로 강조합니다. 포커스된 필드는 `is-active`(테두리·글로우·라벨 색상), 나머지는 `is-dimmed`(opacity·scale 감소)로 구분해 사용자가 현재 입력에만 집중하도록 합니다.
- **GSAP 기반 부드러운 전환:** 엔트리 시 카드·래퍼가 opacity·y·scale로 등장하고, 포커스/블러 시 필드가 GSAP으로 scale(1.02 ↔ 1) 전환되어 부드러운 포커스 이동을 표현합니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호·비밀번호 표시 토글·로그인 버튼만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 480px 이하에서 패딩만 조정합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반. 엔트리·포커스 전환을 GSAP으로 부드럽게 처리합니다.
