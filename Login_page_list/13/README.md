# 로그인 페이지 템플릿 시안 13

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 네오 브루탈리즘  
- **UI 구조:** 풀스크린 몰입형  
- **인터랙션:** 포커스 이동 중심 UX  

---

## 콘셉트 및 UI/UX 특징

- **네오 브루탈리즘:** 굵은 테두리, 높은 대비, 로우한 색감(오프화이트 배경, 검정 테두리, 포인트 컬러)으로 강한 그래픽 톤을 유지합니다.
- **풀스크린 몰입형:** 카드나 사이드 패널 없이 뷰포트 전체를 쓰고, 콘텐츠만 중앙에 배치해 몰입감 있는 단일 화면을 구성합니다.
- **포커스 이동 중심 UX:**  
  - 이메일 → Enter 시 비밀번호 필드로 포커스 이동  
  - 비밀번호 → Enter 시 로그인 제출  
  - 포커스된 필드에 배경/그림자로 상태 강조  
  - 하단 도트로 현재 포커스 단계(이메일/비밀번호) 표시  

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, 로그인 버튼만 제공합니다.
- PC~모바일 반응형: 중앙 정렬과 `clamp` 폰트로 다양한 해상도에서 동일한 톤을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 포커스 상태와 엔트리 애니메이션만 최소한으로 사용합니다.

## 파일 구조

- `index.html` — 로그인 마크업  
- `style.scss` / `style.css` — 시안 전용 스타일  
- `js/login.ts` → `js/login.js` — 로그인 동작(포커스 이동, 제출)  
- `README.md` — 본 설명
