# 로그인 페이지 템플릿 시안 24

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

- **세계관/무드:** 실험적 인터페이스
- **UI 구조:** 풀스크린 몰입형
- **인터랙션:** 포커스 이동 중심 UX

---

## 콘셉트 및 UI/UX 특징

- **실험적 인터페이스:** 다크 배경(#0c0c0e) 위 그리드 패턴, 모노크롬 톤, 시안 계열 포커스(#7dd3fc)로 실험적인 분위기를 냅니다. 단계 인디케이터(1·2)와 포커스별 안내 문구로 입력 흐름을 명시합니다.
- **풀스크린 몰입형:** 뷰포트 전체를 쓰고, 중앙에 폼만 배치해 스크롤 없이 한 화면에서 로그인만 수행합니다.
- **포커스 이동 중심 UX:** 이메일 입력 후 Enter로 비밀번호 필드로 이동, 비밀번호 입력 후 Enter로 제출합니다. `focusStep`에 따라 필드에 `is-focused` 클래스와 단계 네비(1·2)가 갱신되며, 버튼은 두 필드가 채워졌을 때만 `is-ready`로 활성화됩니다.

## 사용 의도 및 구조

- MVP 로그인: 이메일·비밀번호 입력, 비밀번호 표시 토글, Enter 기반 포커스 이동 및 제출만 제공합니다.
- PC~모바일 반응형: 가로·세로 중앙 정렬, 풀스크린 내 고정 플로우로 모든 디바이스에서 동일한 몰입형 레이아웃을 유지합니다.
- HTMX + Alpine.js + TypeScript + SASS/SCSS + GSAP 기반으로, 그리드·내용 엔트리 애니메이션과 포커스 상태 반영을 사용합니다.

- `js/login.ts` → `js/login.js` — 로그인·포커스 이동·GSAP 엔트리
