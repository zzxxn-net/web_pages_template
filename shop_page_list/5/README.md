# 시안 5 — 네오 브루탈리즘 × 마이크로 인터랙션 중심

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 네오 브루탈리즘

UI 구조: 마이크로 인터랙션 중심

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

굵은 테두리·고대비의 네오 브루탈리즘 톤에, 스크롤 없이 한 화면에 고정되는 플로우를 적용했다. 호버·클릭·탭 전환 등 마이크로 인터랙션을 중심으로 GSAP 연출을 넣었다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 100vh 고정 영역으로 스크롤 없는 플로우
- 네오 브루탈: 흰 배경·굵은 검정 테두리·대담한 타이포, 그림자 최소
- 마이크로 인터랙션: 탭 전환·호버 시 테두리/스케일·클릭 시 짧은 피드백
- Alpine.js로 탭·장바구니 상태, GSAP으로 전환·호버 연출

## 사용 의도 및 구조

- `index.html`: 고정 플로우 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 5 전용 스타일(브루탈 톤·고정 레이아웃)
- `js/`: TypeScript 기반 마이크로 인터랙션 및 탭 전환 트리거
