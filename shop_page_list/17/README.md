# 시안 17 — 미래적 클린 × 단일 포커스 입력 흐름

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 미래적 클린

UI 구조: 단일 포커스 입력 흐름

인터랙션: 상태 변화 강조

## 콘셉트

미래적 클린 톤과 단일 포커스 입력 흐름·상태 변화 강조를 결합했다. 한 번에 하나의 상품에만 포커스가 가며, 포커스 변경·장바구니 담김 시 GSAP으로 상태 변화를 강조한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단일 포커스(클릭 시 해당 카드만 강조)
- 미래적 클린: 밝은 배경·정돈된 타이포·블루 악센트
- 단일 포커스 입력 흐름: focusedId 기반 한 카드만 포커스
- 상태 변화 강조: 포커스·담김 시 GSAP 스케일·글로우

## 사용 의도 및 구조

- index.html: 단일 포커스 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 17 전용 스타일(미래적 클린·단일 포커스)
- js/: TypeScript 기반 포커스·상태 변화·GSAP 강조
