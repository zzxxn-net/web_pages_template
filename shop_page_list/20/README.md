# 시안 20 — 실험적 인터페이스 × 단계 분할형 × 포커스 이동 중심

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 실험적 인터페이스

UI 구조: 단계 분할형(Progressive Form)

인터랙션: 포커스 이동 중심 UX

## 콘셉트

실험적 인터페이스 톤과 단계 분할형(Progressive Form), 포커스 이동 중심 UX를 결합했다. 진입 → 상품 탐색 → 장바구니 확인의 3단계로 진행하며, 각 단계 전환 시 GSAP으로 포커스 영역이 이동하는 연출을 적용한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단계별 전환 레이아웃
- 실험적 인터페이스: 기하학적 포인트, 강한 대비 색상
- 단계 분할형: 스텝 인디케이터 + 단계별 콘텐츠 영역
- 포커스 이동 중심: tabindex·focus-visible·GSAP 포커스 하이라이트

## 사용 의도 및 구조

- index.html: 단계형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 20 전용 스타일
- js/: TypeScript 기반 포커스 이동·단계 전환
