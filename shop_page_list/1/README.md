# 시안 1 — 사이버 미니멀 × 카드 분리형

## 적용된 디자인 콘셉트 키워드

- **세계관/무드:** 사이버 미니멀
- **UI 구조:** 카드 분리형 레이아웃
- **인터랙션:** GSAP 기반 부드러운 전환

## 콘셉트

그리드 위주의 카드 분리형 쇼핑 목록으로, 넉넉한 여백과 단순한 톤의 사이버 미니멀 무드를 적용했다. PC~모바일 전 구간 중앙 정렬·반응형 그리드로 일관된 UX를 유지한다.

## UI/UX 특징

- 중앙 정렬 컨테이너 + 카드 그리드 (2~4열 반응형)
- GSAP으로 카드 등장·호버 시 미세 전환
- Alpine.js로 장바구니·좋아요 등 상태 제어
- HTMX로 목록/필터 등 경량 동적 로딩 대응 구조

## 사용 의도 및 구조

- `index.html`: 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 전용 스타일
- `js/`: TypeScript 기반 쇼핑 동작(토글, 애니메이션 트리거)
