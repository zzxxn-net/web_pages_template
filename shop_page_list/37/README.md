# 시안 37 — 감성 테크 × 카드 분리형 레이아웃 × 포커스 이동 중심 UX

## 적용된 디자인 콘셉트 키워드

세계관/무드: 감성 테크
UI 구조: 카드 분리형 레이아웃
인터랙션: 포커스 이동 중심 UX

## 콘셉트

현대 판타지 톤과 마이크로 인터랙션 중심 UI, GSAP 부드러운 전환을 결합했다. 따뜻한 톤과 부드러운 곡선 위에, 카드·버튼·아이콘 등 각 요소의 호버·포커스·클릭마다 GSAP으로 자연스럽게 반응하는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 반응형 카드 그리드
- 감성 테크: 로즈·라벤더·크림 배경, 부드러운 타이포
- 카드 분리형: 각 상품 독립 카드, 여백·테두리로 시각적 분리
- 포커스 이동: 선택 카드 링·스케일 강조, GSAP 포커스 전환

## 사용 의도 및 구조

- index.html: 감성 테크·카드 분리형·포커스 이동 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 37 전용 스타일
- js/: TypeScript 기반 포커스 이동·GSAP 강조 전환
