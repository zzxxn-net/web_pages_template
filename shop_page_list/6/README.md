# 시안 6 — 현대 판타지 × 단일 포커스 입력 흐름

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 현대 판타지

UI 구조: 단일 포커스 입력 흐름

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

부드러운 그라데이션과 은은한 글로우의 현대 판타지 톤에, 한 번에 하나의 포커스(검색 → 목록, 또는 선택 상품)만 강조하는 단일 포커스 입력 흐름을 적용했다. 포커스 전환 시 GSAP으로 부드러운 전환을 넣었다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 판타지 톤 배경·카드
- 단일 포커스: 검색 입력 또는 선택된 상품 하나만 강조, 나머지는 보조
- GSAP: 포커스 이동·상품 선택 시 부드러운 opacity·scale·y 전환
- Alpine.js로 포커스 상태·선택 상품, HTMX 대응 마크업

## 사용 의도 및 구조

- `index.html`: 단일 포커스 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 6 전용 스타일(판타지 톤·포커스 레이아웃)
- `js/`: TypeScript 기반 포커스 전환·GSAP 부드러운 전환 트리거
