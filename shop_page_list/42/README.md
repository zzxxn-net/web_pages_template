# 시안 42 — 미래적 클린 × 비대칭 레이아웃 × GSAP 기반 부드러운 전환

## 적용된 디자인 콘셉트 키워드

세계관/무드: 미래적 클린
UI 구조: 비대칭 레이아웃
인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

미래적 클린 톤과 비대칭 레이아웃, GSAP 기반 부드러운 전환을 결합했다. 밝은 슬레이트·화이트 계열의 클린 배경 위에 히어로 카드(1:1)와 소형 카드 행(1:3)을 비대칭 그리드로 배치하고, 호버·클릭 시 GSAP으로 부드럽게 반응하는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, PC에서 1.2fr / 1fr 비대칭 그리드
- 미래적 클린: #f1f5f9·#ffffff·#3b82f6 액센트, 둥근 radius·미니멀 타이포
- 비대칭 레이아웃: 히어로 1장 + 소형 카드 3장 행(모바일 상단/하단, PC 좌측/우측)
- GSAP 부드러운 전환: 카드·버튼 호버 scale, 장바구니 클릭 시 fromTo 피드백

## 사용 의도 및 구조

- index.html: 미래적 클린·비대칭 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 42 전용 스타일
- js/: TypeScript 기반 GSAP 호버·클릭 전환
