# 시안 13 — 감성 테크 × 비대칭 레이아웃

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 감성 테크

UI 구조: 비대칭 레이아웃

인터랙션: 상태 변화 강조

## 콘셉트

감성 테크 톤과 비대칭 레이아웃을 결합했다. 왼쪽 큰 피처 상품·오른쪽 작은 상품 리스트로 비대칭 그리드를 구성하며, 장바구니 추가·선택 시 상태 변화를 GSAP으로 강조한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 비대칭 그리드(피처 1 + 리스트 N)
- 감성 테크: 따뜻한 로즈·핑크 그라데이션, 부드러운 그림자
- 비대칭: 데스크톱에서 왼쪽 대형 카드·오른쪽 컴팩트 카드
- 상태 변화 강조: 담김·선택 시 GSAP 스케일·펄스·글로우

## 사용 의도 및 구조

- index.html: 비대칭 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 13 전용 스타일(감성 테크·비대칭)
- js/: TypeScript 기반 상태 변화·GSAP 강조
