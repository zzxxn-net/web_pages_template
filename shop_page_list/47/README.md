# 시안 47 — 실험적 인터페이스 × 카드 분리형 레이아웃 × GSAP 기반 부드러운 전환

## 적용된 디자인 콘셉트 키워드

세계관/무드: 실험적 인터페이스
UI 구조: 카드 분리형 레이아웃
인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

실험적 인터페이스 톤과 카드 분리형 레이아웃, GSAP 기반 부드러운 전환을 결합했다. 다크 네이비·코랄 액센트 배경 위에 좌우로 살짝 기울어진 카드 그리드를 두고, 카드 등장·호버(기울기 해제·스케일)·클릭 시 GSAP으로 부드럽게 전환되는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 실험적 컬러(다크 네이비·코랄 액센트)
- 실험적 인터페이스: #16213e·#1a1a2e·#ee6f57·카드 tilt-left/tilt-right
- 카드 분리형 레이아웃: 2×2(모바일)·4열(PC) 그리드·명확한 카드 간격
- GSAP 부드러운 전환: 카드 등장(opacity·y·stagger)·호버 시 rotation 0·scale·클릭 시 링·버튼 스케일
- PC~모바일 반응형 그리드

## 사용 의도 및 구조

- index.html: 실험적·카드 분리형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 47 전용 스타일
- js/: TypeScript 기반 GSAP 부드러운 전환(등장·호버·클릭)
