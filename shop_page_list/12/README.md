# 시안 12 — 현대 판타지 × 단계 분할형

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 현대 판타지

UI 구조: 단계 분할형(Progressive Form)

인터랙션: 입력 반응형 애니메이션

## 콘셉트

현대 판타지 톤과 단계 분할형 플로우를 결합했다. 1단계에서 카테고리를 선택한 뒤 2단계로 넘어가 상품을 보며, 버튼·카드에 호버·클릭 시 입력 반응형 애니메이션으로 피드백을 준다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단계 인디케이터·다음 버튼으로 Progressive Form
- 현대 판타지: 따뜻한 엠버·포레스트 계열, 부드러운 곡선·그림자
- 단계 분할: 1단계 카테고리 선택 → 2단계 상품 그리드
- 입력 반응형: GSAP으로 호버 스케일·클릭 펄스·포커스 글로우

## 사용 의도 및 구조

- index.html: 단계 분할형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 12 전용 스타일(현대 판타지·단계형)
- js/: TypeScript 기반 단계 전환·입력 반응형 GSAP
