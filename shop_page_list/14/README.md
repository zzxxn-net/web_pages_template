# 시안 14 — 네오 브루탈리즘 × 카드 분리형 레이아웃

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 네오 브루탈리즘

UI 구조: 카드 분리형 레이아웃

인터랙션: 포커스 이동 중심 UX

## 콘셉트

네오 브루탈리즘 톤과 카드 분리형 레이아웃을 결합했다. 상품을 명확히 분리된 카드로 배치하고, 탭·포커스 이동 시 GSAP으로 포커스된 요소를 강조해 키보드·순차 탐색 UX를 지원한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 카드 분리형 그리드
- 네오 브루탈리즘: 굵은 테두리·플랫 색·raw 타이포
- 카드 분리형: 각 상품이 독립 카드, 여백·테두리로 분리
- 포커스 이동 중심: tabindex·focus-visible·GSAP 포커스 강조

## 사용 의도 및 구조

- index.html: 카드 분리형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 14 전용 스타일(네오 브루탈·카드 분리)
- js/: TypeScript 기반 포커스 이동·GSAP 포커스 강조
