# 시안 34 — 다크 아카이브 × 단일 포커스 입력 흐름 × GSAP 기반 부드러운 전환

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 다크 아카이브

UI 구조: 단일 포커스 입력 흐름

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

다크 아카이브 톤과 단일 포커스 입력 흐름, GSAP 부드러운 전환을 결합했다. 어두운 배경·스톤 톤 위에 한 번에 하나의 상품만 메인 포커스로 강조하고, 리스트 선택 시 GSAP으로 부드럽게 전환되는 쇼핑 화면이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단일 포커스 메인 + 리스트
- 다크 아카이브: 어두운 배경·스톤 톤·아카이브 느낌 타이포
- 단일 포커스: 메인 1개 상품 강조, 클릭 시 해당 상품으로 포커스 전환
- GSAP: 포커스 전환 시 페이드·스케일 부드러운 전환

## 사용 의도 및 구조

- index.html: 단일 포커스·다크 아카이브 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 34 전용 스타일
- js/: TypeScript 기반 포커스 전환·GSAP 부드러운 전환
