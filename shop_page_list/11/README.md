# 시안 11 — 실험적 인터페이스 × 풀스크린 몰입형

## 적용된 디자인 콘셉트 키워드

세계관/무드: 실험적 인터페이스

UI 구조: 풀스크린 몰입형

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

실험적인 비주얼과 풀스크린 한 화면 몰입 구조를 결합했다. 상품이 화면 전체를 쓰는 풀스크린 슬라이드로 전환되며, GSAP으로 슬라이드·페이드 전환을 부드럽게 연출한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 100vw/100vh 풀스크린 영역
- 실험적 인터페이스: 그라데이션·글래스·비대칭 포지션·실험적 타이포
- 풀스크린 몰입: 한 번에 하나의 상품이 전체 화면, 이전/다음으로 전환
- GSAP: 풀스크린 슬라이드 전환, 요소 등장·퇴장 부드러운 duration/ease

## 사용 의도 및 구조

- index.html: 풀스크린 몰입형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 11 전용 스타일(실험적·풀스크린)
- js/: TypeScript 기반 풀스크린 전환·GSAP 트리거
