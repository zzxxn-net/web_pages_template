# 시안 2 — 다크 아카이브 × 비대칭 레이아웃

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 다크 아카이브

UI 구조: 비대칭 레이아웃

인터랙션: 상태 변화 강조

## 콘셉트

아카이브 감성의 다크 톤과 비대칭 레이아웃으로, 한쪽에 대형 피처 상품·반대쪽에 소형 상품 목록을 배치한다. 장바구니·좋아요 등 상태 변경 시 시각적 피드백을 강조해 사용자 인지도를 높였다.

## UI/UX 특징

- 가로·세로 중앙 정렬 컨테이너, 내부는 비대칭 그리드(피처 1 : 리스트 N)
- PC에서는 좌측 대형 카드·우측 세로 리스트, 모바일에서는 세로 스택
- Alpine.js로 장바구니·좋아요 상태 관리, GSAP으로 상태 변화 시 짧은 애니메이션
- HTMX 대응 마크업으로 추후 목록 갱신 확장 가능

## 사용 의도 및 구조

- `index.html`: 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 2 전용 스타일(비대칭 그리드·다크 아카이브 톤)
- `js/`: TypeScript 기반 상태 변경·애니메이션 트리거
