# 시안 40 — 다크 아카이브 × 풀스크린 몰입형 × GSAP 기반 부드러운 전환

## 적용된 디자인 콘셉트 키워드

세계관/무드: 다크 아카이브
UI 구조: 풀스크린 몰입형
인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

다크 아카이브 톤과 풀스크린 몰입형, GSAP 기반 부드러운 전환을 결합했다. 어두운 배경·스톤 톤 위에 뷰포트 전체를 활용한 몰입형 그리드를 두고, 호버·포커스·클릭 시 GSAP으로 부드럽게 전환되는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 풀스크린 몰입형 그리드
- 다크 아카이브: 어두운 배경·스톤 톤·아카이브 느낌 타이포
- 풀스크린 몰입형: 100vh 중앙 정렬·그리드 4열(PC)·2열(모바일)
- GSAP 부드러운 전환: 호버·포커스·클릭 시 opacity·scale 부드럽게 전환

## 사용 의도 및 구조

- index.html: 다크 아카이브·풀스크린 몰입형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 40 전용 스타일
- js/: TypeScript 기반 GSAP 부드러운 전환(호버·포커스·클릭)
