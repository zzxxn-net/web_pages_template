# 시안 8 — 다크 아카이브 × 단일 포커스 입력 흐름

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 다크 아카이브

UI 구조: 단일 포커스 입력 흐름

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

아카이브 감성의 다크 톤에, 한 번에 하나의 포커스(선택 상품 상세)만 강조하는 단일 포커스 입력 흐름을 적용했다. 포커스 전환 시 GSAP으로 부드러운 전환을 넣었다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 다크 아카이브 배경·카드
- 단일 포커스: 상단/좌측에 선택된 상품 상세 하나만 노출, 목록 클릭 시 해당 상품으로 포커스 전환
- GSAP: 포커스 영역 콘텐츠 교체 시 opacity·y 부드러운 전환
- Alpine.js로 선택 상품·장바구니, HTMX 대응 마크업

## 사용 의도 및 구조

- `index.html`: 단일 포커스 상세 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 8 전용 스타일(다크 아카이브·포커스 패널)
- `js/`: TypeScript 기반 포커스 전환·GSAP 부드러운 전환 트리거
