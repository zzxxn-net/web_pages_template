# 시안 33 — 네오 브루탈리즘 × 단계 분할형(Progressive Form) × 스크롤 없는 고정 플로우

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 네오 브루탈리즘

UI 구조: 단계 분할형(Progressive Form)

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

네오 브루탈리즘 톤과 단계 분할형, 스크롤 없는 고정 플로우를 결합했다. 굵은 테두리·강한 대비 위에 스텝 인디케이터와 단계별 패널을 두고, 100vh 고정 뷰포트에서 스크롤 없이 진행하는 쇼핑 화면이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 고정 뷰포트(스크롤 없음)
- 네오 브루탈리즘: 굵은 테두리·흑백 대비·로우 타이포
- 단계 분할형: 스텝 1·2 인디케이터 + 단계별 콘텐츠 패널
- 스크롤 없는 고정 플로우: 100dvh 고정·overflow hidden

## 사용 의도 및 구조

- index.html: 단계형·고정 플로우 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 33 전용 스타일
- js/: TypeScript 기반 단계 전환·고정 플로우
