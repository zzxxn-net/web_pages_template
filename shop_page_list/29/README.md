# 시안 29 — 실험적 인터페이스 × 마이크로 인터랙션 중심 × 스크롤 없는 고정 플로우

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 실험적 인터페이스

UI 구조: 마이크로 인터랙션 중심

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

실험적 인터페이스 톤과 마이크로 인터랙션 중심, 스크롤 없는 고정 플로우를 결합했다. 한 화면에 모든 상품을 고정 배치하고, 호버·포커스·클릭 시 작은 움직임으로 반응하는 실험적 쇼핑 화면이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 뷰포트 고정(스크롤 없음)
- 실험적 인터페이스: 모노스페이스·강한 테두리·오프셋 그리드
- 마이크로 인터랙션: 호버 시 살짝 기울기·스케일, 포커스 링 펄스, 버튼 클릭 바운스
- 스크롤 없는 고정 플로우: 100vh 내 2×2 그리드로 한 화면 완결

## 사용 의도 및 구조

- index.html: 고정 플로우·마이크로 인터랙션 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 29 전용 스타일
- js/: TypeScript 기반 마이크로 인터랙션(GSAP)
