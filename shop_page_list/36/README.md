# 시안 36 — 현대 판타지 × 마이크로 인터랙션 중심 × GSAP 기반 부드러운 전환

## 적용된 디자인 콘셉트 키워드

세계관/무드: 현대 판타지
UI 구조: 마이크로 인터랙션 중심
인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

현대 판타지 톤과 마이크로 인터랙션 중심 UI, GSAP 부드러운 전환을 결합했다. 따뜻한 톤과 부드러운 곡선 위에, 카드·버튼·아이콘 등 각 요소의 호버·포커스·클릭마다 GSAP으로 자연스럽게 반응하는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 반응형 그리드
- 현대 판타지: 따뜻한 배경·부드러운 radius·판타지 느낌 타이포
- 마이크로 인터랙션: 호버 떠오름·포커스 링·클릭 피드백·장바구니 아이콘 반짝임
- GSAP: 모든 전환을 부드럽게(scale, opacity, filter)

## 사용 의도 및 구조

- index.html: 현대 판타지·마이크로 인터랙션 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 36 전용 스타일
- js/: TypeScript 기반 마이크로 인터랙션·GSAP 부드러운 전환
