# 시안 45 — 현대 판타지 × 비대칭 레이아웃 × 상태 변화 강조

## 적용된 디자인 콘셉트 키워드

세계관/무드: 현대 판타지
UI 구조: 비대칭 레이아웃
인터랙션: 상태 변화 강조

## 콘셉트

현대 판타지 톤과 비대칭 레이아웃, 상태 변화 강조를 결합했다. 따뜻한 베이지·골드 계열 배경 위에 히어로 카드(1개 대형)와 로우(3개 소형) 비대칭 그리드를 두고, 장바구니 담김 시 버튼·배지·카드 링으로 상태 변화를 GSAP으로 강조하는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 현대 판타지 컬러(베이지·골드·담김 그린)
- 현대 판타지: #fdf8f5·#e8d5c4·#c4a574·#6b8e6b 따뜻한 톤
- 비대칭 레이아웃: 히어로 1열 + 로우 3열(소형 카드)
- 상태 변화 강조: 담김 시 배지(개수)·버튼(담김 ✓)·카드 링·GSAP 스케일·백 이즈
- PC~모바일 반응형 그리드

## 사용 의도 및 구조

- index.html: 현대 판타지·비대칭·상태 변화 강조 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 45 전용 스타일
- js/: TypeScript 기반 상태 변화 강조(담김 피드백·배지·GSAP)
