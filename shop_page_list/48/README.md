# 시안 48 — 사이버 미니멀 × 단계 분할형(Progressive Form) × 스크롤 없는 고정 플로우

## 적용된 디자인 콘셉트 키워드

세계관/무드: 사이버 미니멀
UI 구조: 단계 분할형(Progressive Form)
인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

사이버 미니멀 톤과 단계 분할형(Progressive Form), 스크롤 없는 고정 플로우를 결합했다. 슬레이트·블루그레이 배경 위에 1. 상품 보기 / 2. 담기 확인 단계를 헤더에 표시하고, 100dvh 내 2×2 그리드로 한 화면에 고정·스크롤 없이 GSAP 부드러운 호버·클릭 전환을 적용한 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 사이버 미니멀 컬러(#f8fafc·#e2e8f0·#64748b)
- 사이버 미니멀: 미니멀 타이포·슬레이트 그레이·정돈된 카드
- 단계 분할형(Progressive Form): 헤더에 1. 상품 보기(현재)·2. 담기 확인 단계 표시
- 스크롤 없는 고정 플로우: 100vh/100dvh·overflow hidden·2×2 그리드·푸터 담기 확인으로 한 화면 완결
- GSAP: 카드·버튼 호버 스케일·클릭 시 링·스케일 피드백

## 사용 의도 및 구조

- index.html: 사이버 미니멀·단계 분할형·스크롤 없는 고정 플로우 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 48 전용 스타일
- js/: TypeScript 기반 GSAP 부드러운 호버·클릭 전환
