# 시안 16 — 사이버 미니멀 × 풀스크린 몰입형

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 사이버 미니멀

UI 구조: 풀스크린 몰입형

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

사이버 미니멀 톤과 풀스크린 몰입형·스크롤 없는 고정 플로우를 결합했다. 100vh 고정 영역에서 페이지 dot으로만 전환하며, 한 화면에 몰입해 상품을 탐색한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 100vh 고정·스크롤 없음
- 사이버 미니멀: 다크 블루·시안 악센트·미니멀 타이포
- 풀스크린 몰입: 한 화면 고정·페이지 전환으로만 이동
- 스크롤 없는 고정 플로우: 페이지 dot·GSAP 전환

## 사용 의도 및 구조

- index.html: 풀스크린 고정 플로우 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 16 전용 스타일(사이버 미니멀·고정 플로우)
- js/: TypeScript 기반 페이지 전환·GSAP
