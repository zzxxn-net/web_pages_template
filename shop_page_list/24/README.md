# 시안 24 — 감성 테크 × 단계 분할형 × 스크롤 없는 고정 플로우

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 감성 테크

UI 구조: 단계 분할형(Progressive Form)

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

감성 테크 톤과 단계 분할형, 스크롤 없는 고정 플로우를 결합했다. 한 화면 내에서 진입 후 상품 보기 단계로 전환하며, 스크롤 없이 고정 뷰포트에서 진행한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 고정 뷰포트(스크롤 없음)
- 감성 테크: 따뜻한 파스텔·부드러운 색감
- 단계 분할형: 스텝 인디케이터 + 단계별 콘텐츠
- 스크롤 없는 고정 플로우: 100vh 고정·overflow hidden

## 사용 의도 및 구조

- index.html: 단계형·고정 플로우 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 24 전용 스타일
- js/: TypeScript 기반 단계 전환·고정 플로우
