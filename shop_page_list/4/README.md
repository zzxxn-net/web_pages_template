# 시안 4 — 감성 테크 × 풀스크린 몰입형

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 감성 테크

UI 구조: 풀스크린 몰입형

인터랙션: 입력 반응형 애니메이션

## 콘셉트

따뜻한 그라데이션과 부드러운 톤의 감성 테크 무드에, 뷰포트 전체를 쓰는 풀스크린 몰입형 레이아웃을 적용했다. 호버·클릭 등 입력에 반응하는 GSAP 애니메이션으로 몰입감을 높였다.

## UI/UX 특징

- 가로·세로 중앙 정렬, min-height 100vh 풀스크린 영역
- 풀스크린 몰입형: 한 화면에 히어로 상품 + 가로 스크롤/썸네일 스트립으로 몰입감 유지
- 입력 반응형: 호버 시 카드 스케일·그림자, 클릭 시 짧은 펄스 애니메이션
- Alpine.js로 현재 상품·장바구니 상태, GSAP으로 입력 반응 연출

## 사용 의도 및 구조

- `index.html`: 풀스크린 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 4 전용 스타일(감성 톤·풀스크린·반응형)
- `js/`: TypeScript 기반 입력 반응·애니메이션 트리거
