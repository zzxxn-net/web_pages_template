# 시안 38 — 실험적 인터페이스 × 단일 포커스 입력 흐름 × 상태 변화 강조

## 적용된 디자인 콘셉트 키워드

세계관/무드: 실험적 인터페이스
UI 구조: 단일 포커스 입력 흐름
인터랙션: 상태 변화 강조

## 콘셉트

실험적 인터페이스 톤과 단일 포커스 입력 흐름, 상태 변화 강조를 결합했다. 다크 배경·강한 타이포·그린 액센트 위에 메인 상품 1개와 리스트를 두고, 포커스 전환·장바구니 추가 시 GSAP으로 상태 변화를 강조하는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단일 포커스(메인 1개 + 리스트)
- 실험적 인터페이스: 다크 배경·대문자 타이틀·letter-spacing·그린 액센트
- 단일 포커스: 한 번에 하나의 상품만 메인에 표시, 리스트에서 선택 시 전환
- 상태 변화 강조: 포커스 전환·장바구니 클릭 시 GSAP 펄스·하이라이트

## 사용 의도 및 구조

- index.html: 실험적·단일 포커스 쇼핑 페이지 마크업(Alpine.js x-for + 리스트)
- style.scss / style.css: 시안 38 전용 스타일
- js/: TypeScript 기반 단일 포커스 전환·상태 변화 GSAP 강조
