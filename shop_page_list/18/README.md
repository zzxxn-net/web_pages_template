# 시안 18 — 감성 테크 × 단일 포커스 입력 흐름

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 감성 테크

UI 구조: 단일 포커스 입력 흐름

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

감성 테크 톤과 단일 포커스 입력 흐름·GSAP 부드러운 전환을 결합했다. 한 번에 하나의 상품에만 포커스가 가며, 포커스 변경·장바구니 담김 시 GSAP으로 부드럽게 전환한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단일 포커스(클릭 시 해당 카드만 강조)
- 감성 테크: 로즈·핑크 그라데이션, 따뜻한 톤
- 단일 포커스 입력 흐름: focusedId 기반 한 카드만 포커스
- GSAP 부드러운 전환: 포커스·담김 시 duration·ease 연출

## 사용 의도 및 구조

- index.html: 단일 포커스 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 18 전용 스타일(감성 테크·단일 포커스)
- js/: TypeScript 기반 포커스·GSAP 부드러운 전환
