# 시안 19 — 현대 판타지 × 마이크로 인터랙션 중심

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 현대 판타지

UI 구조: 마이크로 인터랙션 중심

인터랙션: 입력 반응형 애니메이션

## 콘셉트

현대 판타지 톤과 마이크로 인터랙션·입력 반응형 애니메이션을 결합했다. 카드·버튼에 호버·클릭 시 GSAP으로 작은 스케일·펄스 등 입력 반응을 연출한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 상품 그리드
- 현대 판타지: 엠버·포레스트 그라데이션, 따뜻한 톤
- 마이크로 인터랙션 중심: 호버·클릭 시 작은 애니메이션
- 입력 반응형: GSAP 호버 스케일·클릭 펄스

## 사용 의도 및 구조

- index.html: 마이크로 인터랙션 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 19 전용 스타일(현대 판타지·마이크로)
- js/: TypeScript 기반 입력 반응형 GSAP
