# 시안 26 — 사이버 미니멀 × 단계 분할형 × 상태 변화 강조

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 사이버 미니멀

UI 구조: 단계 분할형(Progressive Form)

인터랙션: 상태 변화 강조

## 콘셉트

사이버 미니멀 톤과 단계 분할형, 상태 변화 강조를 결합했다. 진입 후 상품 보기 단계로 전환하며, 카드 선택·장바구니 담기 시 상태를 시각적으로 강조한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단계별 패널
- 사이버 미니멀: 다크·시안 톤, 미니멀 타이포
- 단계 분할형: 스텝 인디케이터 + 단계별 콘텐츠
- 상태 변화 강조: 포커스 카드·담김 상태 시각적 강조

## 사용 의도 및 구조

- index.html: 단계형·상태 강조 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 26 전용 스타일
- js/: TypeScript 기반 단계 전환·상태 변화 강조
