# 시안 30 — 미래적 클린 × 풀스크린 몰입형 × 상태 변화 강조

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 미래적 클린

UI 구조: 풀스크린 몰입형

인터랙션: 상태 변화 강조

## 콘셉트

미래적 클린 톤과 풀스크린 몰입형, 상태 변화 강조를 결합했다. 뷰포트 전체를 활용한 몰입형 그리드 위에, 장바구니 담기 시 담김 상태를 시각적으로 강조하는 클린 쇼핑 화면이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 풀스크린 몰입형 레이아웃
- 미래적 클린: 밝은 배경·미니멀 타이포·기하학적 여백
- 풀스크린 몰입형: 100vh 활용·그리드 중앙 몰입
- 상태 변화 강조: 담김 시 버튼·카드 스타일 전환·GSAP 강조

## 사용 의도 및 구조

- index.html: 풀스크린·상태 강조 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 30 전용 스타일
- js/: TypeScript 기반 상태 변화 강조(GSAP)
