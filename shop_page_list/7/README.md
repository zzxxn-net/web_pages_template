# 시안 7 — 실험적 인터페이스 × 카드 분리형 레이아웃

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 실험적 인터페이스

UI 구조: 카드 분리형 레이아웃

인터랙션: 상태 변화 강조

## 콘셉트

비정형 그리드·시각적 실험을 담은 실험적 인터페이스에, 카드 분리형 레이아웃을 적용했다. 장바구니·좋아요 등 상태 변화 시 색·아이콘·GSAP 연출로 강하게 강조한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 실험적 카드 배치(교차·스태거)
- 카드 분리형: 카드 간 여백·테두리로 명확히 분리, 실험적 톤 유지
- 상태 변화 강조: 담김·좋아요 시 버튼 색·문구 변경 및 짧은 GSAP 피드백
- Alpine.js로 장바구니·좋아요 상태, HTMX 대응 마크업

## 사용 의도 및 구조

- `index.html`: 실험적 카드 분리형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 7 전용 스타일(실험 톤·카드 분리)
- `js/`: TypeScript 기반 상태 변화·GSAP 강조 트리거
