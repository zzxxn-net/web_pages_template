# 시안 3 — 미래적 클린 × 단계 분할형(Progressive Form)

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 미래적 클린

UI 구조: 단계 분할형(Progressive Form)

인터랙션: 포커스 이동 중심 UX

## 콘셉트

밝고 정돈된 미래적 클린 톤에, 1단계(카테고리 선택) → 2단계(상품 목록)의 단계 분할형 플로우를 적용했다. 포커스 링·탭 순서·키보드 이동을 명확히 해 포커스 이동 중심 UX를 유지한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 밝은 배경·넉넉한 여백의 클린 레이아웃
- Progressive Form: 1단계에서 카테고리 선택 후 2단계로 전환해 상품 그리드 노출
- 포커스 이동: :focus-visible 강화, 논리적 tab 순서, GSAP으로 포커스 시 미세 하이라이트
- Alpine.js로 단계·선택 상태 관리, HTMX 대응 마크업

## 사용 의도 및 구조

- `index.html`: 단계형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 3 전용 스타일(클린 톤·단계·포커스)
- `js/`: TypeScript 기반 단계 전환·포커스 연동
