# 시안 32 — 사이버 미니멀 × 비대칭 레이아웃 × 입력 반응형 애니메이션

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 사이버 미니멀

UI 구조: 비대칭 레이아웃

인터랙션: 입력 반응형 애니메이션

## 콘셉트

사이버 미니멀 톤과 비대칭 레이아웃, 입력 반응형 애니메이션을 결합했다. 다크·시안 계열 미니멀 위에 왼쪽 대형·오른쪽 소형 카드로 비대칭을 이루고, 호버·포커스·클릭 시 GSAP으로 입력에 반응하는 애니메이션을 적용한 쇼핑 화면이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 비대칭 그리드(대형 1 + 소형 열)
- 사이버 미니멀: 다크 배경·시안 포인트·미니멀 타이포
- 비대칭 레이아웃: 메인 대형 카드 + 서브 소형 카드 열
- 입력 반응형: 호버 스케일·포커스 글로우·클릭 펄스(GSAP)

## 사용 의도 및 구조

- index.html: 비대칭·입력 반응형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 32 전용 스타일
- js/: TypeScript 기반 입력 반응형 GSAP
