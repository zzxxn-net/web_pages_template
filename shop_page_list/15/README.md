# 시안 15 — 다크 아카이브 × 단계 분할형

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 다크 아카이브

UI 구조: 단계 분할형(Progressive Form)

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

다크 아카이브 톤과 단계 분할형 플로우를 결합했다. 1단계에서 카테고리를 선택한 뒤 2단계로 넘어가 상품을 보며, 단계 전환·요소 등장을 GSAP으로 부드럽게 연출한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단계 인디케이터·다음 버튼으로 Progressive Form
- 다크 아카이브: 어두운 배경·아카이브 톤·골드 악센트
- 단계 분할: 1단계 카테고리 선택 → 2단계 상품 그리드
- GSAP: 단계 전환·카드 등장 부드러운 duration·ease

## 사용 의도 및 구조

- index.html: 단계 분할형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 15 전용 스타일(다크 아카이브·단계형)
- js/: TypeScript 기반 단계 전환·GSAP 부드러운 전환
