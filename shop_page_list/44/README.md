# 시안 44 — 다크 아카이브 × 마이크로 인터랙션 중심 × 입력 반응형 애니메이션

## 적용된 디자인 콘셉트 키워드

세계관/무드: 다크 아카이브
UI 구조: 마이크로 인터랙션 중심
인터랙션: 입력 반응형 애니메이션

## 콘셉트

다크 아카이브 톤과 마이크로 인터랙션 중심, 입력 반응형 애니메이션을 결합했다. 어두운 슬레이트·스톤 배경 위에 카드 그리드를 두고, 카드·이미지·버튼 각 요소에 호버·포커스·클릭 시 GSAP으로 입력에 반응하는 마이크로 인터랙션 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 다크 아카이브 컬러(배경·카드·타이포)
- 다크 아카이브: #0f172a·#1e293b·#334155·#94a3b8 스톤 톤
- 마이크로 인터랙션 중심: 카드 호버 스케일·이미지 호버 스케일·버튼 호버·클릭 시 링·스케일 피드백
- 입력 반응형: 호버·포커스·클릭마다 GSAP으로 짧은 duration 전환
- PC 4열·모바일 2열 반응형 그리드

## 사용 의도 및 구조

- index.html: 다크 아카이브·마이크로 인터랙션 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 44 전용 스타일
- js/: TypeScript 기반 입력 반응형 GSAP(카드·이미지·버튼 호버·클릭)
