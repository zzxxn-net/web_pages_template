# 시안 9 — 사이버 미니멀 × 비대칭 레이아웃

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 사이버 미니멀

UI 구조: 비대칭 레이아웃

인터랙션: 포커스 이동 중심 UX

## 콘셉트

넉넉한 여백·단순 톤의 사이버 미니멀 무드에, 좌측 대형·우측 소형의 비대칭 레이아웃을 적용했다. 탭·포커스 순서와 :focus-visible을 명확히 해 포커스 이동 중심 UX를 유지한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 사이버 미니멀 배경·타이포
- 비대칭: PC에서 좌측 2/3 대형 카드·우측 1/3 세로 리스트, 모바일은 세로 스택
- 포커스 이동: 논리적 tab 순서, :focus-visible 링, GSAP으로 포커스 전환 시 미세 전환
- Alpine.js로 선택·장바구니, HTMX 대응 마크업

## 사용 의도 및 구조

- `index.html`: 비대칭 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 9 전용 스타일(사이버 미니멀·비대칭)
- `js/`: TypeScript 기반 포커스 전환·GSAP 트리거
