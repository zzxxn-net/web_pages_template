# 시안 25 — 현대 판타지 × 풀스크린 몰입형 × 상태 변화 강조

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 현대 판타지

UI 구조: 풀스크린 몰입형

인터랙션: 상태 변화 강조

## 콘셉트

현대 판타지 톤과 풀스크린 몰입형 레이아웃, 상태 변화 강조를 결합했다. 한 화면 내 상품 그리드에서 카드 선택·장바구니 담기 시 상태 변화를 시각적으로 강조한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 풀스크린 뷰포트(스크롤 없음)
- 현대 판타지: 엠버·포레스트 그라데이션, 따뜻한 톤
- 풀스크린 몰입형: 100vh 고정·한 화면 내 그리드
- 상태 변화 강조: 포커스 카드·장바구니 담김 상태 시각적 강조

## 사용 의도 및 구조

- index.html: 풀스크린·상태 강조 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 25 전용 스타일
- js/: TypeScript 기반 상태 변화 강조·GSAP
