# 시안 46 — 네오 브루탈리즘 × 단일 포커스 입력 흐름 × 포커스 이동 중심 UX

## 적용된 디자인 콘셉트 키워드

세계관/무드: 네오 브루탈리즘
UI 구조: 단일 포커스 입력 흐름
인터랙션: 포커스 이동 중심 UX

## 콘셉트

네오 브루탈리즘 톤과 단일 포커스 입력 흐름, 포커스 이동 중심 UX를 결합했다. 굵은 검정 테두리·고대비 배경 위에 한 번에 하나의 카드만 포커스(선택)되며, 탭·클릭·호버로 포커스가 이동할 때 GSAP으로 부드럽게 전환되는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 네오 브루탈리즘(검정 테두리·무라운드·고대비)
- 네오 브루탈리즘: 3px 검정 테두리·흑백·굵은 폰트
- 단일 포커스 입력 흐름: focusedId 하나만 유지·카드 클릭/포커스로 전환
- 포커스 이동 중심 UX: tabindex·focus-visible 링·포커스 시 카드 강조(테두리·스케일)·GSAP 전환
- PC 4열·모바일 2열 반응형 그리드

## 사용 의도 및 구조

- index.html: 네오 브루탈리즘·단일 포커스·포커스 이동 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 46 전용 스타일
- js/: TypeScript 기반 focusedId·setFocused·GSAP 포커스 이동 전환
