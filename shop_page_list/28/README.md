# 시안 28 — 감성 테크 × 단일 포커스 입력 흐름 × GSAP 기반 부드러운 전환

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 감성 테크

UI 구조: 단일 포커스 입력 흐름

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

감성 테크 톤과 단일 포커스 입력 흐름, GSAP 부드러운 전환을 결합했다. 한 번에 하나의 상품만 메인 포커스로 강조하며, 상품 선택 시 포커스가 부드럽게 이동하는 흐름을 제공한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 단일 포커스 메인 + 리스트
- 감성 테크: 따뜻한 그라데이션·부드러운 톤·둥근 모서리
- 단일 포커스: 메인 1개 상품 강조, 클릭 시 해당 상품으로 포커스 전환
- GSAP: 포커스 전환·등장 시 페이드·스케일 부드러운 전환

## 사용 의도 및 구조

- index.html: 단일 포커스 흐름 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 28 전용 스타일
- js/: TypeScript 기반 포커스 전환·GSAP 전환
