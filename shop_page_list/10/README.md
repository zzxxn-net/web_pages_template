# 시안 10 — 미래적 클린 × 마이크로 인터랙션 중심

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 미래적 클린

UI 구조: 마이크로 인터랙션 중심

인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

밝고 정돈된 미래적 클린 톤에, 스크롤 없이 한 화면에 고정되는 플로우를 적용했다. 호버·클릭·페이지 전환 등 마이크로 인터랙션을 중심으로 GSAP 연출을 넣었다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 100vh 고정 영역으로 스크롤 없는 플로우
- 미래적 클린: 밝은 배경·넉넉한 여백·단정한 타이포
- 마이크로 인터랙션: 카드 호버 스케일·버튼 클릭 펄스·페이지 점 클릭 시 짧은 전환
- Alpine.js로 페이지·장바구니 상태, GSAP으로 마이크로 전환

## 사용 의도 및 구조

- `index.html`: 고정 플로우 쇼핑 페이지 마크업(HTMX + Alpine.js)
- `style.scss` / `style.css`: 시안 10 전용 스타일(미래적 클린·고정 레이아웃)
- `js/`: TypeScript 기반 마이크로 인터랙션·페이지 전환 트리거
