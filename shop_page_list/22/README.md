# 시안 22 — 네오 브루탈리즘 × 비대칭 레이아웃 × GSAP 기반 부드러운 전환

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 네오 브루탈리즘

UI 구조: 비대칭 레이아웃

인터랙션: GSAP 기반 부드러운 전환

## 콘셉트

네오 브루탈리즘 톤과 비대칭 레이아웃, GSAP 기반 부드러운 전환을 결합했다. 왼쪽 대형 카드·오른쪽 소형 카드로 비대칭 그리드를 구성하고, 카드 등장·호버 시 GSAP으로 부드럽게 전환한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 비대칭 그리드(대형+소형)
- 네오 브루탈리즘: 굵은 테두리·강한 대비·로우한 타이포
- 비대칭 레이아웃: 메인 1열 + 서브 2열
- GSAP 부드러운 전환: 등장·호버·클릭 시 이징 전환

## 사용 의도 및 구조

- index.html: 비대칭 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 22 전용 스타일
- js/: TypeScript 기반 GSAP 부드러운 전환
