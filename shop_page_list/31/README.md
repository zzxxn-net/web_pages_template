# 시안 31 — 현대 판타지 × 카드 분리형 레이아웃 × 포커스 이동 중심 UX

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 현대 판타지

UI 구조: 카드 분리형 레이아웃

인터랙션: 포커스 이동 중심 UX

## 콘셉트

현대 판타지 톤과 카드 분리형 레이아웃, 포커스 이동 중심 UX를 결합했다. 엠버·포레스트 계열 그라데이션 위에 카드를 명확히 분리해 배치하고, 탭·클릭으로 포커스가 이동할 때마다 GSAP으로 포커스 링이 부드럽게 전환되는 쇼핑 화면이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 카드 분리형 그리드
- 현대 판타지: 엠버·포레스트 그라데이션·따뜻한 톤
- 카드 분리형: 명확한 카드 경계·일관된 간격·분리감
- 포커스 이동 중심: 탭/클릭 시 포커스 이동·GSAP 포커스 링 전환

## 사용 의도 및 구조

- index.html: 카드 분리형·포커스 이동 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 31 전용 스타일
- js/: TypeScript 기반 포커스 이동·GSAP 포커스 링
