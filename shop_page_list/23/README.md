# 시안 23 — 미래적 클린 × 카드 분리형 레이아웃 × 입력 반응형 애니메이션

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 미래적 클린

UI 구조: 카드 분리형 레이아웃

인터랙션: 입력 반응형 애니메이션

## 콘셉트

미래적 클린 톤과 카드 분리형 레이아웃, 입력 반응형 애니메이션을 결합했다. 밝고 시원한 색감의 카드 그리드에서 호버·포커스·클릭 시 GSAP으로 카드·버튼이 반응하는 연출을 적용한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 카드 분리형 그리드
- 미래적 클린: 밝은 배경·시원한 액센트·깔끔한 타이포
- 카드 분리형: 명확한 카드 경계·일관된 간격
- 입력 반응형: 호버 스케일·포커스 링·클릭 펄스

## 사용 의도 및 구조

- index.html: 카드 분리형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 23 전용 스타일
- js/: TypeScript 기반 입력 반응형 GSAP
