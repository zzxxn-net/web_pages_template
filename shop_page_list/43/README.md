# 시안 43 — 감성 테크 × 풀스크린 몰입형 × 스크롤 없는 고정 플로우

## 적용된 디자인 콘셉트 키워드

세계관/무드: 감성 테크
UI 구조: 풀스크린 몰입형
인터랙션: 스크롤 없는 고정 플로우

## 콘셉트

감성 테크 톤과 풀스크린 몰입형, 스크롤 없는 고정 플로우를 결합했다. 따뜻한 앰버·골드 그라데이션 배경 위에 100dvh 내 2×2 그리드로 한 화면에 4개 상품을 고정 배치하고, 호버·포커스·클릭 시 GSAP으로 부드럽게 반응하는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 풀스크린 몰입형(100vh/100dvh) + 스크롤 없음
- 감성 테크: 앰버·골드 그라데이션, 반투명 카드·블러, 부드러운 라운드
- 풀스크린 몰입형: 헤더·그리드·푸터가 뷰포트 내에서 flex로 균형 배치
- 스크롤 없는 고정 플로우: 2×2 그리드(4상품)만 표시, overflow: hidden으로 한 화면 완결
- GSAP: 카드 호버 스케일·버튼 호버·클릭 시 링·스케일 피드백

## 사용 의도 및 구조

- index.html: 감성 테크·풀스크린·스크롤 없는 고정 플로우 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 43 전용 스타일
- js/: TypeScript 기반 GSAP 부드러운 전환(호버·클릭)·장바구니 상태
