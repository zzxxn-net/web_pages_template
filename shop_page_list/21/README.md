# 시안 21 — 다크 아카이브 × 풀스크린 몰입형 × 입력 반응형 애니메이션

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 다크 아카이브

UI 구조: 풀스크린 몰입형

인터랙션: 입력 반응형 애니메이션

## 콘셉트

다크 아카이브 톤과 풀스크린 몰입형 레이아웃, 입력 반응형 애니메이션을 결합했다. 한 화면에 상품 그리드를 배치하고, 호버·포커스·클릭 시 GSAP으로 카드·버튼이 반응하는 연출을 적용한다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 풀스크린 뷰포트
- 다크 아카이브: 어두운 배경·세리프 톤·아카이브 느낌 색상
- 풀스크린 몰입형: 스크롤 없이 한 화면 내 그리드·CTA
- 입력 반응형: 호버 스케일·포커스 글로우·클릭 펄스

## 사용 의도 및 구조

- index.html: 풀스크린 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 21 전용 스타일
- js/: TypeScript 기반 입력 반응형 GSAP
