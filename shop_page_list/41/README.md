# 시안 41 — 사이버 미니멀 × 풀스크린 몰입형 × 입력 반응형 애니메이션

## 적용된 디자인 콘셉트 키워드

세계관/무드: 사이버 미니멀
UI 구조: 풀스크린 몰입형
인터랙션: 입력 반응형 애니메이션

## 콘셉트

사이버 미니멀 톤과 풀스크린 몰입형, 입력 반응형 애니메이션을 결합했다. 슬레이트·블루그레이 계열의 미니멀 배경 위에 뷰포트 전체를 활용한 그리드를 두고, 호버·포커스·클릭 시 GSAP으로 입력에 반응하는 쇼핑 페이지이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 풀스크린 몰입형 그리드
- 사이버 미니멀: 슬레이트·블루그레이 배경·미니멀 타이포
- 풀스크린 몰입형: 100vh 중앙 정렬·그리드 4열(PC)·2열(모바일)
- 입력 반응형: 호버·포커스·클릭 시 GSAP 스케일·테두리 반응

## 사용 의도 및 구조

- index.html: 사이버 미니멀·풀스크린 몰입형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 41 전용 스타일
- js/: TypeScript 기반 입력 반응형 GSAP(호버·포커스·클릭)
