# 시안 35 — 미래적 클린 × 풀스크린 몰입형 × 입력 반응형 애니메이션

## 적용된 디자인 콘셉트 키워드

적용된 디자인 콘셉트 키워드:

세계관/무드: 미래적 클린

UI 구조: 풀스크린 몰입형

인터랙션: 입력 반응형 애니메이션

## 콘셉트

미래적 클린 톤과 풀스크린 몰입형, 입력 반응형 애니메이션을 결합했다. 뷰포트 전체를 활용한 몰입형 그리드 위에, 호버·포커스·클릭 시 GSAP으로 입력에 반응하는 애니메이션을 적용한 클린 쇼핑 화면이다.

## UI/UX 특징

- 가로·세로 중앙 정렬, 풀스크린 몰입형 레이아웃
- 미래적 클린: 밝은 배경·미니멀 타이포·기하학적 여백
- 풀스크린 몰입형: 100vh 활용·그리드 중앙 몰입
- 입력 반응형: 호버 스케일·포커스 링·클릭 펄스(GSAP)

## 사용 의도 및 구조

- index.html: 풀스크린·입력 반응형 쇼핑 페이지 마크업(HTMX + Alpine.js)
- style.scss / style.css: 시안 35 전용 스타일
- js/: TypeScript 기반 입력 반응형 GSAP
